### Pattern 01 | 지수·로그 계산: 모양을 만드는 연산 선택, 유리수 지수로 통일

#### 핵심 전략

수능 수학1의 지수·로그 계산 문제(주로 1~2번, 2점)는 다음 두 단계로 해결한다.

**Step 1. 근호를 유리수 지수로 변환한다.**

$$\sqrt[n]{a} = a^{\frac{1}{n}}, \quad \sqrt[n]{a^m} = a^{\frac{m}{n}}$$

모든 거듭제곱근 기호를 유리수 지수 꼴로 바꾸면 지수법칙 적용이 용이해진다.

**Step 2. 밑을 통일한 뒤 지수법칙을 적용한다.**

밑이 다른 경우 소인수분해로 같은 밑으로 만든다.
예: $4^x = (2^2)^x = 2^{2x}$, $8^x = 2^{3x}$

**지수법칙 (실수 지수, $a>0$, $b>0$):**

$$a^m \cdot a^n = a^{m+n}$$
$$a^m \div a^n = a^{m-n}$$
$$(a^m)^n = a^{mn}$$
$$(ab)^n = a^n b^n$$

**로그 연산 법칙 ($a>0$, $a \neq 1$, $M>0$, $N>0$):**

$$\log_a MN = \log_a M + \log_a N$$
$$\log_a \frac{M}{N} = \log_a M - \log_a N$$
$$\log_a M^k = k \log_a M$$
$$\log_a b = \frac{\log_c b}{\log_c a} \quad \text{(밑 변환 공식)}$$
$$\log_a b \cdot \log_b c = \log_a c \quad \text{(연쇄 로그)}$$

#### 출제 유형 및 예시

**유형 A: 지수 계산 (2점)**

> $\sqrt[4]{5} \times 5^{\frac{3}{4}}$ 의 값을 구하시오.

**풀이 전략:**
$$\sqrt[4]{5} \times 5^{\frac{3}{4}} = 5^{\frac{1}{4}} \times 5^{\frac{3}{4}} = 5^{\frac{1}{4}+\frac{3}{4}} = 5^1 = 5$$

**유형 B: 로그 계산 (3점)**

> $\log_2 3 = a$, $\log_2 5 = b$ 일 때, $\log_2 30$을 $a$, $b$로 나타내시오.

**풀이 전략:**
$$\log_2 30 = \log_2 (2 \cdot 3 \cdot 5) = 1 + \log_2 3 + \log_2 5 = 1 + a + b$$

**유형 C: 혼합 계산 (3점)**

> $3^a = 2$, $3^b = 5$ 일 때, $6^{\frac{1}{a}} \cdot 15^{\frac{1}{b}}$의 값을 구하시오.

**풀이 전략:**
$a = \log_3 2$, $b = \log_3 5$임을 이용.
$\frac{1}{a} = \log_2 3$, $\frac{1}{b} = \log_5 3$으로 변환하여 계산.

#### AI 풀이 시 주의사항
- 문제에 $\sqrt{}$, $\sqrt[n]{}$ 기호가 있으면 **즉시 유리수 지수로 변환**한다.
- 밑이 서로 다른 경우 소인수분해로 **공통 밑을 찾는 것**이 1순위 전략이다.
- 로그 계산에서 $\log_a b \cdot \log_b c = \log_a c$ 연쇄 공식을 적극 활용한다.

### Pattern 02 | 거듭제곱근: 3가지 동치 표현 암기

#### 핵심 전략

다음 3가지 표현이 **완전히 동일한 의미**임을 이해하고, 문제에 따라 변환하여 사용한다.

$$\boxed{a\text{의 } n\text{제곱근}} \iff \boxed{n\text{제곱하여 } a\text{가 되는 수}} \iff \boxed{\text{방정식 } x^n = a\text{의 근}}$$

**실근의 개수 판별:**

| | $a > 0$ | $a = 0$ | $a < 0$ |
|---|---|---|---|
| $n$이 짝수 | 실근 2개 ($\pm \sqrt[n]{a}$) | 실근 1개 ($0$) | 실근 0개 |
| $n$이 홀수 | 실근 1개 ($\sqrt[n]{a}$) | 실근 1개 ($0$) | 실근 1개 ($\sqrt[n]{a}$) |

**거듭제곱근 기호 약속:**
- $\sqrt[n]{a}$: $n$제곱근 중 **양의 실수** ($n$이 짝수이고 $a>0$일 때)
- $\sqrt[n]{a}$: $n$제곱근 중 **유일한 실수** ($n$이 홀수일 때)

**핵심 공식:**

$$(\sqrt[n]{a})^n = a \quad (a>0)$$
$$\sqrt[n]{a^n} = a \quad (a>0)$$
$$\sqrt[m]{\sqrt[n]{a}} = \sqrt[mn]{a} = a^{\frac{1}{mn}}$$

#### 출제 유형 및 예시

**유형 A: 거듭제곱근의 개수 (3점)**

> $-8$의 세제곱근 중 실수인 것을 모두 구하시오.

**풀이 전략:**
$x^3 = -8$의 실근을 구하면 됨. $n=3$은 홀수이고 $a=-8<0$이므로 실근 1개.
$x = \sqrt[3]{-8} = -2$

**유형 B: 거듭제곱근 계산 (3점)**

> $\sqrt[3]{\sqrt{a^6}}$을 간단히 하시오. (단, $a>0$)

**풀이 전략:**
$$\sqrt[3]{\sqrt{a^6}} = \sqrt[3]{a^3} = a$$

또는: $a^6$을 유리수 지수로: $\sqrt[3]{(a^6)^{\frac{1}{2}}} = \sqrt[3]{a^3} = a$

#### AI 풀이 시 주의사항
- $\sqrt[n]{a}$와 "$a$의 $n$제곱근"은 다르다. 전자는 하나의 값, 후자는 집합 개념.
- 실근 개수 판별 시 $n$의 홀짝, $a$의 부호 **두 가지**를 반드시 확인한다.

### Pattern 03 | 지수·로그 방정식·부등식: 밑 통일

#### 핵심 전략

지수·로그 방정식·부등식의 **유일한 전략은 밑 통일**이다.

**Step 1. 밑을 통일한다.**

지수방정식: $2^x = 4^{x-1}$ → $2^x = 2^{2(x-1)}$ (밑을 2로 통일)

로그방정식: $\log_2 x = \log_4 (x+2)$ → 밑 변환 공식으로 같은 밑으로 통일

**Step 2. 지수(또는 로그값)를 비교한다.**

- $a > 0$, $a \neq 1$일 때: $a^f = a^g \iff f = g$
- $a > 1$일 때: $a^f < a^g \iff f < g$ (단조증가)
- $0 < a < 1$일 때: $a^f < a^g \iff f > g$ (단조감소)

**Step 3. 치환을 활용한다.**

$t = a^x$ 또는 $t = \log_a x$로 치환하면 이차방정식 형태로 변환되는 경우가 많다.

**로그 방정식 풀이 시 반드시 진수 조건 확인:**

$$\log_a f(x) = k \implies f(x) > 0 \text{ 확인 필수}$$

#### 출제 유형 및 예시

**유형 A: 지수방정식 (3점)**

> $4^x - 3 \cdot 2^{x+1} + 8 = 0$을 풀어라.

**풀이 전략:**
$t = 2^x$ ($t>0$)로 치환:
$$t^2 - 6t + 8 = 0 \implies (t-2)(t-4) = 0$$
$t = 2$ 또는 $t = 4$이므로 $x = 1$ 또는 $x = 2$

**유형 B: 로그부등식 (4점)**

> $\log_{\frac{1}{2}}(x-1) > \log_{\frac{1}{2}}(5-x)$를 풀어라.

**풀이 전략:**
- 진수 조건: $x - 1 > 0$이고 $5 - x > 0$ → $1 < x < 5$
- 밑이 $\frac{1}{2} < 1$이므로 부등호 방향 반전: $x - 1 < 5 - x$
- $2x < 6$ → $x < 3$
- 진수 조건과 교집합: $1 < x < 3$

#### AI 풀이 시 주의사항
- 로그 문제에서 **진수 조건 체크는 필수**이며, 최종 답에서 반드시 대입 검증한다.
- 지수부등식에서 밑이 $0 < a < 1$인 경우 **부등호 방향이 반전**됨을 주의한다.
- 치환 후 새 변수의 범위($t > 0$ 등)를 항상 명시한다.

### Pattern 04 | 지수·로그 함수의 그래프: 점근선, 역함수 관계

#### 핵심 전략

지수함수 $y = a^x$와 로그함수 $y = \log_a x$의 핵심 성질:

| 성질 | 지수함수 $y = a^x$ | 로그함수 $y = \log_a x$ |
|---|---|---|
| 정의역 | 실수 전체 | $x > 0$ |
| 치역 | $y > 0$ | 실수 전체 |
| 점근선 | $x$축 ($y = 0$) | $y$축 ($x = 0$) |
| 지나는 점 | $(0, 1)$ | $(1, 0)$ |
| 역함수 | $y = \log_a x$ | $y = a^x$ |

**평행이동한 함수의 성질 변화:**

$y = a^{x-p} + q$:
- 점근선: $y = q$
- 지나는 점: $(p, 1+q)$
- 정의역: 실수 전체, 치역: $y > q$

$y = \log_a (x - p) + q$:
- 점근선: $x = p$
- 지나는 점: $(p+1, q)$
- 정의역: $x > p$

**역함수 관계:** $y = a^x$와 $y = \log_a x$는 **직선 $y = x$에 대해 대칭**이다.

#### 출제 유형 및 예시

**유형 A: 점근선과 좌표 (3점)**

> 함수 $y = 2^{x-3} + 1$의 점근선의 방정식과 $y$절편을 구하시오.

**풀이 전략:**
- 점근선: $y = 1$
- $y$절편: $x = 0$ 대입 → $y = 2^{-3} + 1 = \frac{1}{8} + 1 = \frac{9}{8}$

**유형 B: 역함수 활용 (4점)**

> 함수 $f(x) = \log_3(x+a) + b$의 역함수가 $g(x) = 3^{x-2} - 1$일 때, $a + b$의 값을 구하시오.

**풀이 전략:**
$g(x)$의 점근선은 $y = -1$, 지나는 점은 $(2, 0)$.
역함수 관계이므로 $f(x)$의 점근선은 $x = -1$, 지나는 점은 $(0, 2)$.
→ $a = 1$, $b = 2$ → $a + b = 3$

#### AI 풀이 시 주의사항
- 평행이동 후 **점근선·정의역·치역이 모두 바뀜**을 항상 확인한다.
- 역함수 관계는 $x$, $y$를 서로 교환하는 것임을 이용해 빠르게 계산한다.
- 그래프 문제에서 점근선의 방정식을 먼저 구하면 대부분의 정보가 따라온다.

### Pattern 05 | 삼각함수 값: 하나를 알면 나머지를 모두 구한다

#### 핵심 전략

삼각함수 하나의 값과 각의 범위가 주어지면 나머지 모든 삼각함수 값을 구할 수 있다.

**피타고라스 항등식:**

$$\sin^2\theta + \cos^2\theta = 1$$
$$1 + \tan^2\theta = \sec^2\theta$$
$$1 + \cot^2\theta = \csc^2\theta$$

**삼각함수의 부호 (각의 범위로 사분면 판별):**

| 사분면 | $\sin\theta$ | $\cos\theta$ | $\tan\theta$ |
|---|---|---|---|
| 1사분면 | $+$ | $+$ | $+$ |
| 2사분면 | $+$ | $-$ | $-$ |
| 3사분면 | $-$ | $-$ | $+$ |
| 4사분면 | $-$ | $+$ | $-$ |

**변환 공식:**

$$\sin(-\theta) = -\sin\theta, \quad \cos(-\theta) = \cos\theta, \quad \tan(-\theta) = -\tan\theta$$
$$\sin(\pi - \theta) = \sin\theta, \quad \cos(\pi - \theta) = -\cos\theta$$
$$\sin\left(\frac{\pi}{2} - \theta\right) = \cos\theta, \quad \cos\left(\frac{\pi}{2} - \theta\right) = \sin\theta$$

#### 출제 유형 및 예시

**유형 A: 삼각함수 값 계산 (3점)**

> $\frac{\pi}{2} < \theta < \pi$이고 $\sin\theta = \frac{\sqrt{3}}{2}$일 때, $\cos\theta$와 $\tan\theta$를 구하시오.

**풀이 전략:**
2사분면이므로 $\cos\theta < 0$.
$$\cos\theta = -\sqrt{1 - \sin^2\theta} = -\sqrt{1 - \frac{3}{4}} = -\frac{1}{2}$$
$$\tan\theta = \frac{\sin\theta}{\cos\theta} = \frac{\frac{\sqrt{3}}{2}}{-\frac{1}{2}} = -\sqrt{3}$$

**유형 B: 변환 공식 활용 (3점)**

> $\sin\left(-\theta\right) = \frac{1}{3}$일 때, $\cos\left(\frac{\pi}{2} + \theta\right)$의 값을 구하시오.

**풀이 전략:**
$$\sin(-\theta) = -\sin\theta = \frac{1}{3} \implies \sin\theta = -\frac{1}{3}$$
$$\cos\left(\frac{\pi}{2} + \theta\right) = -\sin\theta = \frac{1}{3}$$

#### AI 풀이 시 주의사항
- 각의 범위로 **사분면을 먼저 결정**하고 부호를 확정한 뒤 계산한다.
- $\sin(-\theta)$, $\cos(\pi - \theta)$ 등 변환 공식을 기계적으로 암기하여 빠르게 처리한다.
- $\tan\theta = \frac{\sin\theta}{\cos\theta}$를 활용하면 별도 계산 없이 $\tan$ 값을 구할 수 있다.

### Pattern 06 | 삼각함수 그래프: 주기·진폭·평행이동 분석

#### 핵심 전략

$y = a\sin(bx + c) + d$ 형태의 그래프는 다음 4가지 요소로 완전히 결정된다.

| 요소 | 공식 | 의미 |
|---|---|---|
| 진폭 | $\|a\|$ | 최댓값 - 최솟값의 절반 |
| 주기 | $\frac{2\pi}{\|b\|}$ | 한 주기의 길이 |
| 위상 | $-\frac{c}{b}$ | $x$축 방향 평행이동량 |
| 수직이동 | $d$ | $y$축 방향 평행이동량 |

**최댓값·최솟값:**
$$\text{최댓값} = |a| + d, \quad \text{최솟값} = -|a| + d$$

**2가지 핵심 풀이법:**

**방법 1. 그래프 분석법:** 그래프의 최댓값, 최솟값, 주기를 읽어 $a$, $b$, $d$를 역산한다.

**방법 2. 특정 좌표 대입법:** 그래프가 지나는 특정 점의 좌표를 방정식에 대입하여 계수를 결정한다.

#### 출제 유형 및 예시

**유형 A: 그래프에서 값 읽기 (3점)**

> 함수 $f(x) = a\sin(bx) + c$의 그래프에서 최댓값이 5, 최솟값이 1, 주기가 $\pi$일 때, $a$, $b$, $c$를 구하시오. (단, $a > 0$, $b > 0$)

**풀이 전략:**
- 진폭: $a = \frac{5-1}{2} = 2$
- 수직이동: $c = \frac{5+1}{2} = 3$
- 주기: $\frac{2\pi}{b} = \pi \implies b = 2$

**유형 B: 방정식의 실근 개수 (4점)**

> $0 \leq x \leq 2\pi$에서 방정식 $\sin x = k$의 실근의 개수를 $k$ 값에 따라 구하시오.

**풀이 전략:**
$y = \sin x$의 그래프와 직선 $y = k$의 교점 수를 시각화:
- $k > 1$ 또는 $k < -1$: 0개
- $k = 1$ 또는 $k = -1$: 1개
- $-1 < k < 1$ ($k \neq 0$): 2개
- $k = 0$: $x = 0, \pi, 2\pi$이므로 3개 (구간 설정에 따라 다를 수 있음)

#### AI 풀이 시 주의사항
- 그래프 문제에서는 **주기·진폭·최댓값·최솟값을 먼저 파악**하는 것이 핵심이다.
- $y = a\cos(bx+c)+d$도 동일한 전략이 적용된다.
- 방정식 실근 개수 문제는 반드시 **그래프를 그려서 교점 수**로 판단한다.

### Pattern 07 | 삼각함수의 활용: 사인법칙·코사인법칙 키워드 매핑

#### 핵심 전략

삼각형 $ABC$에서 다음 공식을 **키워드에 따라 자동으로 선택**한다.

**사인법칙:**

$$\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R$$

($R$: 외접원 반지름)

**사인법칙 사용 키워드:**
- "외접원 반지름"이 등장하면 → **무조건 사인법칙**
- 변과 그 대각이 함께 주어지면 → **사인법칙**
- 두 각과 한 변이 주어지면 → **사인법칙**

**코사인법칙:**

$$a^2 = b^2 + c^2 - 2bc\cos A$$
$$\cos A = \frac{b^2 + c^2 - a^2}{2bc}$$

**코사인법칙 사용 키워드:**
- 세 변이 모두 주어지면 → **코사인법칙으로 각 계산**
- 두 변과 끼인각이 주어지면 → **코사인법칙으로 나머지 변 계산**
- 변과 각이 섞여 있는데 외접원이 없으면 → **코사인법칙 우선 시도**

**삼각형의 넓이:**

$$S = \frac{1}{2}ab\sin C = \frac{1}{2}bc\sin A = \frac{1}{2}ca\sin B$$
$$S = \frac{abc}{4R} \quad \text{(외접원 반지름 활용)}$$

#### 출제 유형 및 예시

**유형 A: 사인법칙 활용 (3점)**

> 삼각형 $ABC$에서 $\angle A = 60°$, $a = \sqrt{3}$일 때, 외접원의 반지름 $R$을 구하시오.

**풀이 전략:**
$$2R = \frac{a}{\sin A} = \frac{\sqrt{3}}{\sin 60°} = \frac{\sqrt{3}}{\frac{\sqrt{3}}{2}} = 2 \implies R = 1$$

**유형 B: 코사인법칙 활용 (4점)**

> 삼각형 $ABC$에서 $a = 7$, $b = 5$, $c = 3$일 때, $\cos A$를 구하시오.

**풀이 전략:**
$$\cos A = \frac{b^2 + c^2 - a^2}{2bc} = \frac{25 + 9 - 49}{2 \cdot 5 \cdot 3} = \frac{-15}{30} = -\frac{1}{2}$$

**유형 C: 두 법칙 혼합 (4점)**

> 삼각형 $ABC$에서 $\angle B = 60°$, $b = \sqrt{7}$, $c = 2$일 때, $a$를 구하시오.

**풀이 전략:**
$b$와 그 대각 $B$가 주어졌지만 구하는 것이 $a$이므로 코사인법칙 사용:
$$b^2 = a^2 + c^2 - 2ac\cos B$$
$$7 = a^2 + 4 - 2a \cdot 2 \cdot \frac{1}{2} = a^2 - 2a + 4$$
$$a^2 - 2a - 3 = 0 \implies (a-3)(a+1) = 0$$
$a > 0$이므로 $a = 3$

#### AI 풀이 시 주의사항
- **"외접원"이라는 단어가 나오면 사인법칙**, "세 변" 또는 "두 변과 끼인각"이면 코사인법칙으로 자동 판단한다.
- 넓이가 나오는 경우 $S = \frac{1}{2}ab\sin C$와 사인법칙·코사인법칙을 연립한다.
- 수능에서는 두 법칙을 모두 활용하는 혼합형 문제가 자주 출제된다.

### Pattern 08 | 등차·등비수열: 공식 암기 + 항 간 관계 속산

#### 핵심 전략

**등차수열 공식:**

$$a_n = a_1 + (n-1)d$$
$$S_n = \frac{n(a_1 + a_n)}{2} = na_1 + \frac{n(n-1)}{2}d$$

**등차중항:** 세 수 $a$, $b$, $c$가 등차수열 $\iff$ $2b = a + c$

**등비수열 공식:**

$$a_n = a_1 \cdot r^{n-1}$$
$$S_n = \frac{a_1(r^n - 1)}{r - 1} \quad (r \neq 1), \quad S_n = na_1 \quad (r = 1)$$

**등비중항:** 세 수 $a$, $b$, $c$가 등비수열 $\iff$ $b^2 = ac$

**속산 전략 (항 간 관계):**

일반항을 직접 구하지 않고도 다음 관계를 이용해 빠르게 계산:
- 등차수열: $a_{m} - a_{n} = (m-n)d$
- 등비수열: $\frac{a_m}{a_n} = r^{m-n}$
- $S_n$과 $a_n$의 관계: $a_n = S_n - S_{n-1}$ ($n \geq 2$), $a_1 = S_1$

#### 출제 유형 및 예시

**유형 A: 등차수열 일반항 (3점)**

> 등차수열 $\{a_n\}$에 대하여 $a_3 = 7$, $a_7 = 19$일 때, $a_{10}$을 구하시오.

**풀이 전략:**
$a_7 - a_3 = 4d = 12 \implies d = 3$
$a_1 = a_3 - 2d = 7 - 6 = 1$
$a_{10} = 1 + 9 \cdot 3 = 28$

**유형 B: 등비수열 조건 (3점)**

> 첫째항과 공비가 모두 양수인 등비수열 $\{a_n\}$에 대하여 $\frac{a_4}{a_2} + \frac{a_2}{a_1} = 30$일 때, 공비 $r$을 구하시오.

**풀이 전략:**
$\frac{a_4}{a_2} = r^2$, $\frac{a_2}{a_1} = r$이므로:
$$r^2 + r = 30 \implies r^2 + r - 30 = 0 \implies (r+6)(r-5) = 0$$
$r > 0$이므로 $r = 5$

**유형 C: $S_n$과 $a_n$ 관계 (4점)**

> 수열 $\{a_n\}$의 첫째항부터 제$n$항까지의 합 $S_n = 2n^2 + 3n$일 때, $a_5$를 구하시오.

**풀이 전략:**
$a_5 = S_5 - S_4 = (50 + 15) - (32 + 12) = 65 - 44 = 21$

검증: $a_1 = S_1 = 5$, $a_n = S_n - S_{n-1} = 4n + 1$ ($n \geq 2$), $a_5 = 21$ ✓

#### AI 풀이 시 주의사항
- $S_n$이 주어진 경우 $a_n = S_n - S_{n-1}$ 공식을 사용하되 **$n=1$일 때 반드시 별도 검증**한다.
- 등비수열에서 공비를 구할 때 $r > 0$ 조건이 주어지면 음수 해를 버린다.
- 등차·등비수열 혼합 조건에서는 $a_n$의 두 표현을 연립해서 풀면 된다.

### Pattern 09 | 수열의 합: 4가지 유형 분류 + 나열 전략

#### 핵심 전략

$\sum$ (시그마) 계산은 다음 4가지 유형 중 하나로 분류한다.

**유형 1. 기본 시그마 공식 직접 적용:**

$$\sum_{k=1}^{n} k = \frac{n(n+1)}{2}$$
$$\sum_{k=1}^{n} k^2 = \frac{n(n+1)(2n+1)}{6}$$
$$\sum_{k=1}^{n} k^3 = \left(\frac{n(n+1)}{2}\right)^2$$
$$\sum_{k=1}^{n} c = cn, \quad \sum_{k=1}^{n} (af(k) \pm bg(k)) = a\sum f(k) \pm b\sum g(k)$$

**유형 2. 등차수열의 합으로 변환:**

일반항 $a_k$가 $k$의 일차식이면 → 등차수열 합 공식 적용

**유형 3. 등비수열의 합으로 변환:**

일반항 $a_k$가 $r^k$ 형태이면 → 등비수열 합 공식 적용

$$\sum_{k=1}^{n} r^k = \frac{r(r^n - 1)}{r - 1} \quad (r \neq 1)$$

**유형 4. 위 3가지로 해결 안 될 때 → 나열(직접 계산):**

$n$이 특정 작은 값이거나, 규칙성이 보이면 항을 직접 나열하여 합을 구한다.

**분수 꼴 일반항의 분리 (부분분수):**

$$\frac{1}{k(k+1)} = \frac{1}{k} - \frac{1}{k+1}$$
$$\frac{1}{k(k+d)} = \frac{1}{d}\left(\frac{1}{k} - \frac{1}{k+d}\right)$$

#### 출제 유형 및 예시

**유형 A: 기본 시그마 (3점)**

> $\sum_{k=1}^{10}(3k^2 - 2k + 1)$을 계산하시오.

**풀이 전략:**
$$3 \cdot \frac{10 \cdot 11 \cdot 21}{6} - 2 \cdot \frac{10 \cdot 11}{2} + 10 = 3 \cdot 385 - 110 + 10 = 1155 - 100 = 1055$$

**유형 B: 부분분수 (3점)**

> $\sum_{k=1}^{n} \frac{1}{k(k+2)}$를 구하시오.

**풀이 전략:**
$$\frac{1}{k(k+2)} = \frac{1}{2}\left(\frac{1}{k} - \frac{1}{k+2}\right)$$
텔레스코핑(망원급수):
$$= \frac{1}{2}\left(1 + \frac{1}{2} - \frac{1}{n+1} - \frac{1}{n+2}\right)$$

**유형 C: $\sum$과 일반항 연결 (4점)**

> $\sum_{k=1}^{n} a_k = n^2 + 2n$일 때, $a_{10}$을 구하시오.

**풀이 전략:**
$a_n = S_n - S_{n-1} = (n^2 + 2n) - ((n-1)^2 + 2(n-1)) = 2n + 1$ ($n \geq 2$)
$a_1 = S_1 = 3 = 2(1) + 1$ → 공식 성립
$a_{10} = 2(10) + 1 = 21$

#### AI 풀이 시 주의사항
- 분수 꼴 일반항이 나오면 **부분분수로 변환 후 텔레스코핑**이 거의 항상 정답.
- $\sum_{k=1}^{n} a_k = S_n$으로 두고 $a_n = S_n - S_{n-1}$ 관계를 활용한다.
- 일반항이 복잡하면 **먼저 몇 항을 나열**하여 패턴을 파악한 뒤 접근한다.

### Pattern 10 | 수학적 귀납법·점화식: 나열 후 규칙 발견, $n$ 자리에 $n \pm 1$ 대입

#### 핵심 전략

**점화식 문제 풀이 전략:**

**Step 1. 항을 직접 나열한다.**

$a_1$, $a_2$, $a_3$, ...을 점화식에 따라 순서대로 계산하여 패턴을 파악한다.

**Step 2. $n$ 자리에 $n+1$ 또는 $n-1$을 대입하여 연립한다.**

예: $a_{n+1} = f(a_n)$ 꼴에서 $n$ 대신 $n-1$을 대입하면 $a_n = f(a_{n-1})$을 얻어 연립 가능.

**주요 점화식 유형과 변환:**

| 점화식 | 변환 방법 |
|---|---|
| $a_{n+1} = a_n + d$ | 등차수열: $a_n = a_1 + (n-1)d$ |
| $a_{n+1} = r \cdot a_n$ | 등비수열: $a_n = a_1 \cdot r^{n-1}$ |
| $a_{n+1} = a_n + f(n)$ | 양변 합산: $a_n = a_1 + \sum_{k=1}^{n-1} f(k)$ |
| $a_{n+1} = p \cdot a_n + q$ | $b_n = a_n + \frac{q}{p-1}$로 치환 → 등비수열 |

**수학적 귀납법 증명 구조:**

1. **기저 단계:** $n = 1$일 때 명제 $P(1)$이 성립함을 보인다.
2. **귀납 단계:** $P(n)$이 성립한다고 가정할 때 $P(n+1)$이 성립함을 보인다.

#### 출제 유형 및 예시

**유형 A: 점화식으로 일반항 구하기 (3점)**

> 수열 $\{a_n\}$이 $a_1 = 2$, $a_{n+1} = 2a_n + 3$을 만족할 때, $a_n$을 구하시오.

**풀이 전략:**
$b_n = a_n + 3$으로 치환:
$$b_{n+1} = a_{n+1} + 3 = 2a_n + 6 = 2(a_n + 3) = 2b_n$$
$b_1 = a_1 + 3 = 5$, $\{b_n\}$은 초항 5, 공비 2인 등비수열.
$$b_n = 5 \cdot 2^{n-1} \implies a_n = 5 \cdot 2^{n-1} - 3$$

**유형 B: 조건부 점화식 (4점)**

> 수열 $\{a_n\}$이 $a_1 = 3$이고 다음을 만족시킨다.

$$a_{n+1} = \begin{cases} a_n - 3 & (|a_n|\text{이 홀수}) \\ \dfrac{1}{2}a_n & (a_n = 0 \text{ 또는 } |a_n|\text{이 짝수}) \end{cases}$$

> $a_6 + a_7 = 3$이 되도록 하는 $a_1$의 값을 구하시오.

**풀이 전략:**
$a_1$부터 차례로 나열하며 조건을 추적한다.

$a_1 = 3$ (홀수) → $a_2 = 0$ → $a_3 = 0$ → ... (고정)

역추적: $a_6 + a_7 = 3$이 되는 경우를 목표로 $a_5$, $a_4$, ... 역산하거나 여러 $a_1$ 후보를 나열하여 탐색.

나열 원칙: 홀수이면 $-3$, 짝수·0이면 $\div 2$를 반복 적용.

**유형 C: 수학적 귀납법 (4점)**

> 모든 자연수 $n$에 대하여 $\sum_{k=1}^{n} k = \frac{n(n+1)}{2}$임을 수학적 귀납법으로 증명하시오.

**풀이 전략:**
- **$n=1$:** 좌변 $= 1$, 우변 $= \frac{1 \cdot 2}{2} = 1$. 성립.
- **귀납 가정:** $\sum_{k=1}^{n} k = \frac{n(n+1)}{2}$가 성립한다고 가정.
- **$n+1$:** $\sum_{k=1}^{n+1} k = \frac{n(n+1)}{2} + (n+1) = \frac{(n+1)(n+2)}{2}$. 성립.

#### AI 풀이 시 주의사항
- 조건부 점화식($|a_n|$의 홀짝에 따라 규칙이 다른 경우) 문제는 **항을 직접 나열**하는 것이 가장 빠르다.
- $a_{n+1} = pa_n + q$ 꼴은 $a_n + c = p(a_{n-1} + c)$ 꼴로 변환하는 **치환 기법**을 활용한다.
- 귀납법 증명에서는 귀납 단계에서 **가정한 식을 명시적으로 사용**해야 논리가 완결된다.

### B1 | 등차수열 핵심 유형 (Pattern 08 연결)

#### 출제 경향 분석

등차수열 문제는 크게 3가지 유형으로 반복 출제된다.

| 유형 | 조건 형태 | 핵심 전략 |
|---|---|---|
| 유형 1 | $a_m$, $a_n$ 두 항 값이 주어짐 | 연립방정식으로 $a_1$, $d$ 결정 |
| 유형 2 | $S_n$ 조건이 포함됨 | $S_n = \frac{n(a_1+a_n)}{2}$ 또는 $a_n = S_n - S_{n-1}$ 활용 |
| 유형 3 | 부등식 조건 ($S_n > 0$, $a_n < 0$ 등) | 일반항 공식으로 부등식 설정 후 풀이 |

---

#### B1 유형 1 | 두 조건으로 수열 결정

**문제 구조:**

등차수열 $\{a_n\}$에 대하여 두 조건이 주어질 때, 특정 항 또는 합을 구한다.

**대표 예시:**

> 등차수열 $\{a_n\}$에 대하여 $a_2 + a_6 = 20$, $a_4 = 8$일 때, 공차 $d$와 $a_{10}$을 구하시오.

**풀이 전략 및 사고 흐름:**

```
Step 1. $a_n = a_1 + (n-1)d$로 각 조건을 $a_1$, $d$의 식으로 표현
  → $a_2 + a_6 = (a_1 + d) + (a_1 + 5d) = 2a_1 + 6d = 20$
  → $a_4 = a_1 + 3d = 8$

Step 2. 연립방정식 풀기
  → $2a_1 + 6d = 20$과 $2(a_1 + 3d) = 16$이 동일 → 정보 부족
  → 추가 조건 활용 또는 다른 해석 필요

※ 실전 주의: 두 조건이 사실상 같은 정보를 담고 있을 수 있다.
   이 경우 매개변수로 표현하거나 추가 조건을 찾는다.

Step 3. 구하는 값 계산
  → $a_{10} = a_1 + 9d$에 대입
```

**등차수열 연립 풀이 핵심 공식:**

두 항 $a_m$, $a_n$이 주어지면:
$$d = \frac{a_m - a_n}{m - n}, \quad a_1 = a_n - (n-1)d$$

---

#### B1 유형 2 | $S_n$ 조건 포함

**문제 구조:**

첫째항 또는 합 $S_n$과 관련된 조건이 주어질 때 항 또는 합을 구한다.

**대표 예시:**

> 첫째항이 $3$인 등차수열 $\{a_n\}$의 첫째항부터 제$n$항까지의 합을 $S_n$이라 하자.
> $S_5 = S_{10}$일 때, $S_{15}$를 구하시오.

**풀이 전략 및 사고 흐름:**

```
Step 1. $S_n = na_1 + \frac{n(n-1)}{2}d$에 $a_1 = 3$ 대입
  → $S_5 = 15 + 10d$
  → $S_{10} = 30 + 45d$

Step 2. $S_5 = S_{10}$ 조건 적용
  → $15 + 10d = 30 + 45d$
  → $-35d = 15$ → $d = -\frac{3}{7}$

Step 3. $S_{15}$ 계산
  → $S_{15} = 45 + 105 \cdot (-\frac{3}{7}) = 45 - 45 = 0$

※ 패턴 인식: $S_m = S_n$ $(m \neq n)$이면 $S_{m+n} = 0$이 성립한다.
   (수능에서 자주 활용되는 등차수열의 성질)
```

**핵심 성질 (수능 빈출):**

$$S_m = S_n \ (m \neq n) \implies S_{m+n} = 0$$

#### B1 유형 3 | 부등식 조건

**문제 구조:**

등차수열에서 $a_n > 0$, $S_n > 0$ 등의 부등식을 만족하는 $n$의 최솟값 또는 최댓값을 구한다.

**대표 예시:**

> 등차수열 $\{a_n\}$에 대하여 $a_3 = 20$, $a_7 = 8$일 때,
> $S_n > 0$을 만족시키는 자연수 $n$의 최댓값을 구하시오.

**풀이 전략 및 사고 흐름:**

```
Step 1. $d$와 $a_1$ 결정
  → $d = \frac{8-20}{7-3} = -3$, $a_1 = 20 - 2(-3) = 26$

Step 2. $S_n$ 공식으로 부등식 설정
  → $S_n = 26n + \frac{n(n-1)}{2}(-3) = n\left(26 - \frac{3(n-1)}{2}\right) > 0$
  → $n > 0$이므로 $26 - \frac{3(n-1)}{2} > 0$
  → $52 > 3(n-1)$ → $n < 18.3...$

Step 3. 최댓값 결정
  → 최댓값 $n = 18$
```

### B2 | 등비수열 핵심 유형 (Pattern 08 연결)

#### 출제 경향 분석

| 유형 | 조건 형태 | 핵심 전략 |
|---|---|---|
| 유형 1 | 두 항의 비 또는 곱 조건 | $a_m / a_n = r^{m-n}$ 활용 |
| 유형 2 | $S_n$ 조건 | $S_n - S_{n-1} = a_n$ 또는 등비급수 공식 |
| 유형 3 | 등비·등차 혼합 조건 | 두 조건을 연립하여 $a_1$, $r$ 결정 |

---

#### B2 유형 1 | 항 사이의 관계

**대표 예시:**

> 첫째항과 공비가 모두 양수인 등비수열 $\{a_n\}$이
> $\dfrac{a_4}{a_2} + \dfrac{a_2}{a_1} = 30$을 만족시킬 때, 공비 $k$의 값을 구하시오.

**풀이 전략 및 사고 흐름:**

```
Step 1. 항 간 비를 공비로 표현
  → $\frac{a_4}{a_2} = r^2$, $\frac{a_2}{a_1} = r$

Step 2. 방정식 설정
  → $r^2 + r = 30$
  → $r^2 + r - 30 = 0$
  → $(r+6)(r-5) = 0$

Step 3. 조건 검토
  → $r > 0$이므로 $r = 5$
```

**핵심 공식:**

$$\frac{a_m}{a_n} = r^{m-n}, \quad a_m \cdot a_n = a_k^2 \text{ (등비중항, } k = \frac{m+n}{2}\text{)}$$

---

#### B2 유형 2 | $S_n$ 조건 포함

**대표 예시:**

> 등비수열 $\{a_n\}$의 첫째항부터 제$n$항까지의 합을 $S_n$이라 하자.
> $S_4 - S_2 = 3a_4$, $a_5 = \dfrac{3}{4}$일 때, $a_1 + a_2$의 값을 구하시오.

**풀이 전략 및 사고 흐름:**

```
Step 1. $S_4 - S_2 = a_3 + a_4$임을 이용
  → $a_3 + a_4 = 3a_4$
  → $a_3 = 2a_4$

Step 2. 공비 결정
  → $\frac{a_3}{a_4} = \frac{1}{r} = 2$ → $r = \frac{1}{2}$

Step 3. $a_1$ 결정
  → $a_5 = a_1 \cdot r^4 = a_1 \cdot \frac{1}{16} = \frac{3}{4}$
  → $a_1 = 12$

Step 4. $a_1 + a_2$ 계산
  → $a_1 + a_2 = 12 + 6 = 18$
```

#### B2 유형 3 | 등비·등차 혼합

**대표 예시:**

> 세 수 $a-2$, $a$, $3a-4$가 이 순서대로 등비수열을 이루고,
> 세 수 $a-2$, $a$, $b$가 이 순서대로 등차수열을 이룰 때, $b$를 구하시오.

**풀이 전략 및 사고 흐름:**

```
Step 1. 등비중항 조건 적용
  → $a^2 = (a-2)(3a-4) = 3a^2 - 10a + 8$
  → $2a^2 - 10a + 8 = 0$ → $a^2 - 5a + 4 = 0$
  → $(a-1)(a-4) = 0$ → $a = 1$ 또는 $a = 4$

Step 2. 등비수열 공비 확인 (공비 ≠ 0 검증)
  → $a = 1$: 세 수 $-1, 1, -1$ → 공비 $-1$ (가능)
  → $a = 4$: 세 수 $2, 4, 8$ → 공비 $2$ (가능)

Step 3. 등차중항 조건으로 $b$ 결정
  → $a = 4$일 때: $2b = 2 + 4 = 6$ → $b = 3$ (또는 $a=1$ 경우도 계산)
```

### B3 | 수열의 합 핵심 유형 (Pattern 09 연결)

#### 출제 경향 분석

| 유형 | 조건 형태 | 핵심 전략 |
|---|---|---|
| 유형 1 | $\sum$ 직접 계산 | 기본 공식 조합 |
| 유형 2 | $\sum a_k = f(n)$ 꼴 | $a_n = S_n - S_{n-1}$ |
| 유형 3 | 분수 꼴 $a_k$ | 부분분수 → 텔레스코핑 |
| 유형 4 | 점화식 + 합 | 점화식으로 $a_n$ 구한 후 합 계산 |

---

#### B3 유형 1 | 시그마 직접 계산

**대표 예시:**

> 수열 $\{a_n\}$에 대하여 $a_n = 2n - 1$일 때, $\displaystyle\sum_{k=1}^{n}(a_k + 3)$을 구하시오.

**풀이 전략 및 사고 흐름:**

```
Step 1. 시그마 분리
  → $\sum_{k=1}^{n}(a_k + 3) = \sum_{k=1}^{n}a_k + \sum_{k=1}^{n}3$

Step 2. 각각 계산
  → $\sum_{k=1}^{n}(2k-1) = 2 \cdot \frac{n(n+1)}{2} - n = n^2$
  → $\sum_{k=1}^{n}3 = 3n$

Step 3. 합산
  → $n^2 + 3n$
```

---

#### B3 유형 2 | $\sum a_k = f(n)$으로 $a_n$ 구하기

**대표 예시:**

> 수열 $\{a_n\}$에 대하여 $\displaystyle\sum_{k=1}^{n}a_k = 3n^2 - n$일 때, $a_{10}$을 구하시오.

**풀이 전략 및 사고 흐름:**

```
Step 1. $n \geq 2$일 때 $a_n = S_n - S_{n-1}$ 적용
  → $a_n = (3n^2 - n) - (3(n-1)^2 - (n-1))$
  → $= 3n^2 - n - 3n^2 + 6n - 3 + n - 1$
  → $= 6n - 4$

Step 2. $n = 1$ 검증
  → $a_1 = S_1 = 3(1)^2 - 1 = 2$
  → 공식 $6(1) - 4 = 2$ ✓ (성립하므로 $n \geq 1$에서 $a_n = 6n - 4$)

Step 3. $a_{10}$ 계산
  → $a_{10} = 6(10) - 4 = 56$
```

---

#### B3 유형 3 | 부분분수와 텔레스코핑

**대표 예시:**

> $\displaystyle\sum_{k=1}^{5}\frac{1}{a_k \cdot a_{k+1}} = \frac{5}{96}$이고 공차가 $0$이 아닌 등차수열 $\{a_n\}$에 대하여
> $\displaystyle\sum_{k=1}^{15}a_k$를 구하시오.

**풀이 전략 및 사고 흐름:**

```
Step 1. 부분분수 변환
  → $\frac{1}{a_k \cdot a_{k+1}} = \frac{1}{d}\left(\frac{1}{a_k} - \frac{1}{a_{k+1}}\right)$

Step 2. 텔레스코핑으로 합 계산
  → $\sum_{k=1}^{5}\frac{1}{a_k a_{k+1}}
     = \frac{1}{d}\left(\frac{1}{a_1} - \frac{1}{a_6}\right)
     = \frac{5}{96}$

Step 3. 조건을 $a_1$, $d$로 표현하여 연립
  → $a_6 = a_1 + 5d$를 대입
  → $\frac{5}{a_1(a_1+5d)} = \frac{5}{96}$ → $a_1(a_1+5d) = 96$

Step 4. 추가 조건으로 $a_1$, $d$ 결정 후 $\sum_{k=1}^{15}a_k$ 계산
  → $S_{15} = \frac{15(a_1 + a_{15})}{2} = \frac{15(2a_1 + 14d)}{2}$
```

**핵심 공식 정리:**

$$\frac{1}{a_k a_{k+1}} = \frac{1}{d}\left(\frac{1}{a_k} - \frac{1}{a_{k+1}}\right) \quad \text{(공차 } d \text{인 등차수열)}$$

$$\sum_{k=1}^{n}\frac{1}{k(k+1)} = 1 - \frac{1}{n+1} = \frac{n}{n+1}$$

#### B3 유형 4 | 점화식 + 합

**대표 예시:**

> 수열 $\{a_n\}$이 모든 자연수 $n$에 대하여
> $a_{n+4} + a_n = 12$를 만족시킬 때, $\displaystyle\sum_{n=1}^{16}a_n$의 값을 구하시오.

**풀이 전략 및 사고 흐름:**

```
Step 1. 점화식에서 주기성 파악
  → $a_{n+4} = 12 - a_n$이므로
  → $a_{n+8} = 12 - a_{n+4} = 12 - (12 - a_n) = a_n$
  → 주기가 8임을 확인

Step 2. 한 주기 합 계산
  → $a_1 + a_2 + ... + a_8$의 합 계산
  → $a_{n+4} + a_n = 12$에서
     $(a_1+a_5) + (a_2+a_6) + (a_3+a_7) + (a_4+a_8) = 48$

Step 3. 전체 합 계산
  → $\sum_{n=1}^{16}a_n = 2 \times 48 = 96$

※ 핵심 인식: 점화식에서 $a_{n+k} + a_n = c$ 꼴이 나오면 주기성 탐색이 1순위
```

### B3 심화 | 수학적 귀납법·조건부 점화식 핵심 유형 (Pattern 10 연결)

#### 출제 경향 분석

수열 단원에서 가장 어려운 유형은 **조건부 점화식**이다.
$|a_n|$의 홀짝이나 $a_n$의 부호에 따라 다른 규칙이 적용되며, 역추적이 핵심이다.

---

#### 조건부 점화식 풀이 원칙

**원칙 1. 목표값에서 역추적한다.**

$a_6 + a_7 = k$가 되는 $a_1$을 구하는 문제 → $a_6$, $a_7$의 가능한 값 조합을 먼저 나열하고, 각각에서 역방향으로 $a_1$을 추적한다.

**원칙 2. 순방향 나열로 궤도를 파악한다.**

가능한 $a_1$ 후보를 설정하고 점화식을 앞에서부터 적용하여 $a_6$, $a_7$이 조건을 만족하는지 확인한다.

**원칙 3. 수렴·순환 구간을 파악한다.**

조건부 점화식은 일정 시점 이후 값이 $0$으로 수렴하거나 일정 주기로 순환하는 경우가 많다. 이 구간을 빠르게 파악하면 역추적 범위를 줄일 수 있다.

---

#### 대표 예시: 2025학년도 수능 22번 구조 유형

**문제 구조:**

> 모든 항이 정수이고 다음 조건을 만족시키는 모든 수열 $\{a_n\}$에 대하여 $|a_1|$의 값의 합을 구하시오.
>
> **(가)** 모든 자연수 $n$에 대하여
>
> $$a_{n+1} = \begin{cases} a_n - 3 & (|a_n|\text{이 홀수인 경우}) \\ \dfrac{1}{2}a_n & (a_n = 0 \text{ 또는 } |a_n|\text{이 짝수인 경우}) \end{cases}$$
>
> **(나)** $|a_m| = |a_{m+2}|$인 자연수 $m$의 최솟값은 3이다.

**풀이 전략 및 사고 흐름:**

```
[조건 분석]
- 조건 (나): $|a_m| = |a_{m+2}|$인 최솟값이 3
  → $m = 1$, $m = 2$에서는 $|a_m| \neq |a_{m+2}|$
  → $m = 3$에서 처음으로 $|a_3| = |a_5|$

[규칙 정리]
- $|a_n|$이 홀수: $a_{n+1} = a_n - 3$ (홀수 → 짝수)
- $|a_n|$이 짝수 또는 0: $a_{n+1} = \frac{1}{2}a_n$ (크기 감소)

[궤도 분석]
- $a_n = 0$: 이후 계속 0 (흡수 상태)
- 홀수 → $-3$ → 짝수 → $\div 2$ 반복

[조건 (나) 만족 조건]
- $m=1$: $|a_1| \neq |a_3|$ 이어야 함
- $m=2$: $|a_2| \neq |a_4|$ 이어야 함
- $m=3$: $|a_3| = |a_5|$ 이어야 함

[탐색 전략]
1. $a_1$의 홀짝에 따라 $a_2$ 결정
2. $a_2$의 홀짝에 따라 $a_3$ 결정
3. $m=1, 2$에서 조건 불만족, $m=3$에서 만족하는 케이스를 체계적으로 탐색
4. 조건을 만족하는 모든 $a_1$ 수집 후 $|a_1|$ 합산
```

|---|---|
| $\sqrt[n]{a^n} = a$ 로 착각 | $n$이 짝수이면 $\sqrt[n]{a^n} = \|a\|$ |
| 로그 방정식 풀고 끝 | 반드시 진수 조건 대입 검증 |
| $\log_a b < 0$ 인데 부등호 방향 유지 | 밑 $a < 1$이면 부등호 반전 |
| $\log_a (f+g) = \log_a f + \log_a g$ | 틀림: $\log_a (fg) = \log_a f + \log_a g$ |

### 삼각함수

| 함정 | 올바른 처리 |
|---|---|
| $\sin^2\theta + \cos^2\theta = 1$ 공식 후 부호 실수 | 사분면으로 부호 먼저 결정 |
| 주기를 $2\pi$ 로만 생각 | $y = \sin(bx)$의 주기는 $\frac{2\pi}{b}$ |
| 코사인 법칙에서 각도 순서 혼동 | $a^2 = b^2 + c^2 - 2bc\cos A$: $a$의 대각이 $A$ |

### 수열

| 함정 | 올바른 처리 |
|---|---|
| $S_n$ 공식에서 $n=1$ 생략 | $a_n = S_n - S_{n-1}$은 $n \geq 2$, $a_1 = S_1$ 별도 |
| 등비수열에서 $r < 0$ 가능성 간과 | 조건에 "양수"가 없으면 음수 공비도 검토 |
| $\sum_{k=1}^{n}c = cn$ 으로 처리 | 맞음 (단, $\sum_{k=3}^{n}c = c(n-2)$처럼 범위 주의) |
| 점화식에서 $n=1$ 예외 미확인 | 일반항 도출 후 $n=1$ 대입 검증 필수 |

### Pattern 01 Part2 | 지수·로그 계산: 목표값을 먼저 정하고 역산

#### 수능·평가원 핵심 유형 분류

| 세부 유형 | 조건 형태 | 접근 전략 |
|---|---|---|
| 유형 1 | $\log a$, $\log b$ 주어지고 $\log f(a,b)$ 요구 | 목표 식에서 필요한 연산 역산 |
| 유형 2 | $a^x = k$ 꼴로 주어지고 지수 조합 요구 | 지수 사이 관계 파악 후 조합 |
| 유형 3 | 정수 조건 포함 (자연수, 정수 범위 탐색) | 부등식으로 범위 설정 후 열거 |

---

#### 유형 1 | 로그 연산 역산

**핵심 사고 원칙:**

> 구하는 값에 무엇이 필요한지 먼저 본 뒤, 그것을 만들기 위한 연산을 선택한다.

**대표 예시:**

> $1$보다 큰 두 실수 $a$, $b$가
> $\log a + \log b = 3$, $\log a \cdot \log b = 2$를 만족시킬 때,
> $\log\dfrac{\sqrt{a}}{b}$의 값을 구하시오.

**풀이 전략 및 사고 흐름:**

```
Step 1. 목표값 분석
  → $\log\frac{\sqrt{a}}{b} = \frac{1}{2}\log a - \log b$
  → $\log a$와 $\log b$ 각각의 값이 필요

Step 2. 조건에서 $\log a$, $\log b$ 역산
  → $\log a + \log b = 3$, $\log a \cdot \log b = 2$
  → $p = \log a$, $q = \log b$로 치환하면
  → $p + q = 3$, $pq = 2$ → 이차방정식 $t^2 - 3t + 2 = 0$
  → $(t-1)(t-2) = 0$ → $p, q$는 $1$ 또는 $2$

Step 3. 조건 검토 후 계산
  → $a > 1$, $b > 1$이므로 $\log a > 0$, $\log b > 0$ 둘 다 만족
  → 두 가지 경우:
     (i) $\log a = 2$, $\log b = 1$ → $\frac{1}{2}(2) - 1 = 0$
     (ii) $\log a = 1$, $\log b = 2$ → $\frac{1}{2}(1) - 2 = -\frac{3}{2}$
  → 추가 조건이 없으면 두 값 모두 고려
```

**핵심 공식 재확인:**

$$\log a + \log b = k,\ \log a \cdot \log b = m
\implies \log a,\ \log b \text{는 } t^2 - kt + m = 0 \text{의 두 근}$$

---

#### 유형 2 | 지수 조건 → 지수 조합

**대표 예시:**

> 두 양수 $a$, $b$가 $\log_3 a = \log_9 b$를 만족시킬 때,
> $\log_3 a + \log_3 b$를 $\log_3 a$로 나타내시오.

**풀이 전략 및 사고 흐름:**

```
Step 1. 밑 통일로 조건 정리
  → $\log_9 b = \frac{\log_3 b}{2}$이므로
  → $\log_3 a = \frac{\log_3 b}{2}$ → $\log_3 b = 2\log_3 a$

Step 2. 목표값 계산
  → $\log_3 a + \log_3 b = \log_3 a + 2\log_3 a = 3\log_3 a$
```

---

#### 유형 3 | 정수 조건 + 로그 범위 탐색

**대표 예시:**

> $\log_2(x-3)$이 정의되고 $\log_4(3x-5)$와 같아지는 자연수 $x$를 모두 구하시오.

**풀이 전략 및 사고 흐름:**

```
Step 1. 정의 조건(진수 조건) 확인
  → $x - 3 > 0$ → $x > 3$
  → $3x - 5 > 0$ → $x > \frac{5}{3}$
  → 교집합: $x > 3$ → 자연수이므로 $x \geq 4$

Step 2. 밑 통일 후 방정식 풀기
  → $\log_4(3x-5) = \frac{\log_2(3x-5)}{2}$
  → $\log_2(x-3) = \frac{\log_2(3x-5)}{2}$
  → $2\log_2(x-3) = \log_2(3x-5)$
  → $(x-3)^2 = 3x - 5$
  → $x^2 - 9x + 14 = 0$ → $(x-2)(x-7) = 0$

Step 3. 범위 조건 교집합
  → $x = 7$ ($x = 2$는 $x > 3$ 불만족)
```

### Pattern 02 Part2 | 거듭제곱근: 방정식 실근 개수 응용

#### 수능·평가원 핵심 유형 분류

| 세부 유형 | 조건 형태 | 접근 전략 |
|---|---|---|
| 유형 1 | $f(n)$의 $n$제곱근 중 실수 개수 탐색 | $n$의 홀짝 + $f(n)$ 부호 판별 |
| 유형 2 | 어떤 자연수의 $k$제곱근이 되는 경우 탐색 | $x^k = N$ 형태로 변환 후 정수 조건 |
| 유형 3 | 이차함수와 거듭제곱근 결합 | 방정식 실근 수와 중근 조건 연계 |

---

#### 유형 1 | 실수 개수 탐색

**대표 예시:**

> $1 \leq n \leq 20$인 자연수 $n$에 대하여 $n$의 세제곱근 중 실수인 것의 개수를 $f(n)$이라 할 때,
> $\displaystyle\sum_{n=1}^{20} f(n)$을 구하시오.

**풀이 전략 및 사고 흐름:**

```
[규칙 정리]
- $n$의 세제곱근: 방정식 $x^3 = n$의 실근
- $k = 3$ (홀수), $n > 0$ → 실근 항상 1개

→ 모든 자연수 $n$ (양수)에 대해 세제곱근의 실수는 항상 1개
→ $f(n) = 1$ for all $n = 1, \ldots, 20$
→ $\sum_{n=1}^{20} f(n) = 20$
```

**홀수 $k$와 짝수 $k$ 비교 핵심 표:**

| $n$의 부호 | $k$ 홀수 | $k$ 짝수 |
|---|---|---|
| $n > 0$ | 실수 1개 ($\sqrt[k]{n}$) | 실수 2개 ($\pm\sqrt[k]{n}$) |
| $n = 0$ | 실수 1개 ($0$) | 실수 1개 ($0$) |
| $n < 0$ | 실수 1개 ($\sqrt[k]{n} < 0$) | 실수 0개 |

---

#### 유형 2 | "어떤 자연수의 $k$제곱근" 조건

**대표 예시:**

> $n$이 어떤 자연수의 네제곱근이 되도록 하는 자연수 $n$의 개수를 $1 \leq n \leq 100$ 범위에서 구하시오.

**풀이 전략 및 사고 흐름:**

```
Step 1. 조건 변환
  → "$n$이 어떤 자연수 $m$의 네제곱근" ⟺ $n^4 = m$ (단, $m$은 자연수)
  → $n$이 자연수이면 $n^4$도 자연수 → 조건은 자동 성립

Step 2. 범위 탐색
  → $1 \leq n \leq 100$의 모든 자연수 $n$에 대해 $n^4$는 자연수
  → 즉, 범위 내 모든 자연수 100개 모두 해당

※ "어떤 자연수의 네제곱근 중 실수" 조건이면:
  → 네제곱근 중 실수는 $\pm\sqrt[4]{m}$
  → $n > 0$이어야 하므로 양의 실수 조건 추가
```

### Pattern 03 Part2 | 지수·로그 방정식·부등식 심화

#### 수능·평가원 핵심 유형 분류

| 세부 유형 | 대표 형태 | 전략 |
|---|---|---|
| 유형 1 | 지수방정식 → 이차방정식 치환 | $t = a^x$으로 치환 후 이차방정식 |
| 유형 2 | 로그방정식 두 근의 합·곱 | 근과 계수의 관계 활용 |
| 유형 3 | 연립 지수·로그 방정식 | 로그 취하여 연립 |
| 유형 4 | 좌표평면 위 두 점 조건 | 두 로그의 관계를 기울기 조건으로 변환 |

---

#### 유형 1 | 이차방정식 치환

**대표 예시:**

> 방정식 $4^x - 3 \cdot 2^{x+1} + 8 = 0$의 모든 실근의 합을 구하시오.

**풀이 전략 및 사고 흐름:**

```
Step 1. 치환
  → $t = 2^x$ ($t > 0$)
  → $4^x = (2^x)^2 = t^2$, $3 \cdot 2^{x+1} = 6 \cdot 2^x = 6t$
  → $t^2 - 6t + 8 = 0$

Step 2. 이차방정식 풀기
  → $(t-2)(t-4) = 0$ → $t = 2$ 또는 $t = 4$

Step 3. $x$ 역산
  → $2^x = 2$ → $x = 1$
  → $2^x = 4$ → $x = 2$

Step 4. 모든 실근의 합 = $1 + 2 = 3$
```

---

#### 유형 2 | 로그방정식 두 근의 합·곱

**대표 예시:**

> 로그방정식 $(\log x)^2 - 3\log x + 2 = 0$의 두 근을 $\alpha$, $\beta$라 할 때,
> $\alpha \cdot \beta$의 값을 구하시오.

**풀이 전략 및 사고 흐름:**

```
Step 1. 치환
  → $t = \log x$로 치환
  → $t^2 - 3t + 2 = 0$ → $(t-1)(t-2) = 0$
  → $t = 1$ 또는 $t = 2$

Step 2. $x$ 역산
  → $\log x = 1$ → $x = 10$
  → $\log x = 2$ → $x = 100$

Step 3. 진수 조건 확인
  → $x = 10 > 0$, $x = 100 > 0$ → 모두 유효

Step 4. $\alpha \cdot \beta = 10 \times 100 = 1000$

※ 빠른 방법: 두 근 $t_1 = 1$, $t_2 = 2$에서
  $\log\alpha + \log\beta = t_1 + t_2 = 3$
  → $\log(\alpha\beta) = 3$ → $\alpha\beta = 10^3 = 1000$
```

---

#### 유형 3 | 연립 지수·로그 방정식

**대표 예시:**

> 연립방정식
> $\begin{cases} 2^x \cdot 4^y = 8 \\ 3^x \cdot 9^y = 27 \end{cases}$
> 의 해 $x$, $y$에 대하여 $x + y$의 값을 구하시오.

**풀이 전략 및 사고 흐름:**

```
Step 1. 밑 통일
  → $2^x \cdot 2^{2y} = 2^3$ → $x + 2y = 3$
  → $3^x \cdot 3^{2y} = 3^3$ → $x + 2y = 3$

※ 두 식이 동일 → 추가 조건 필요
  → 이 경우 매개변수로 일반해 표현

Step 2. 독립 조건이 있는 경우 연립
  → $x + 2y = 3$ 조건 하에서 $x + y$는 $y$에 따라 달라짐
  → 추가 조건 $y = 1$ 등 주어지면 $x = 1$, $x + y = 2$
```

---

#### 유형 4 | 좌표평면 + 로그 조건

**대표 예시:**

> 두 양수 $a$, $b$에 대하여 두 점 $(\log a, \log b)$, $(2\log a, \log 3b)$를 지나는 직선이 원점을 지날 때, $\log\dfrac{a}{b}$의 값을 구하시오.

**풀이 전략 및 사고 흐름:**

```
Step 1. 원점을 지나는 조건 설정
  → 원점 $(0,0)$, 점 $(\log a, \log b)$를 지나는 직선의 기울기:
  → $k = \frac{\log b}{\log a} = \frac{\log b - 0}{\log a - 0}$

Step 2. 두 번째 점도 같은 직선 위 조건
  → $\frac{\log 3b}{2\log a} = \frac{\log b}{\log a}$
  → $\log 3b = 2\log b$
  → $\log 3 + \log b = 2\log b$
  → $\log 3 = \log b$ → $b = 3$

Step 3. $a$ 조건은 자유이나 $\log(a/b) = \log a - \log 3$
  → 문제에 따라 추가 조건으로 $a$ 결정
```

### Pattern 04 Part2 | 지수·로그함수 그래프 심화

#### 수능·평가원 핵심 유형 분류

| 세부 유형 | 대표 형태 | 전략 |
|---|---|---|
| 유형 1 | 두 함수 교점 조건 | 연립방정식 → 교점 $x$ 좌표 |
| 유형 2 | 점근선·역함수·대칭 관계 | 점근선과 역함수 좌표 교환 |
| 유형 3 | 그래프와 정수·자연수 조건 결합 | 그래프 위 범위 탐색 |

---

#### 유형 1 | 두 함수의 교점

**대표 예시:**

> 두 곡선 $y = \log_2(x+2)$와 $y = \log_4(3x+4)$가 만나는 점의 $x$좌표를 구하시오.

**풀이 전략 및 사고 흐름:**

```
Step 1. 밑 통일
  → $\log_4(3x+4) = \frac{\log_2(3x+4)}{2}$
  → $\log_2(x+2) = \frac{\log_2(3x+4)}{2}$
  → $2\log_2(x+2) = \log_2(3x+4)$
  → $(x+2)^2 = 3x+4$

Step 2. 방정식 풀기
  → $x^2 + 4x + 4 = 3x + 4$
  → $x^2 + x = 0$ → $x(x+1) = 0$
  → $x = 0$ 또는 $x = -1$

Step 3. 진수 조건
  → $x + 2 > 0$, $3x + 4 > 0$ 모두 만족하는 값
  → $x = 0$: $2 > 0$, $4 > 0$ ✓
  → $x = -1$: $1 > 0$, $1 > 0$ ✓ (둘 다 유효)
```

---

#### 유형 2 | 역함수·대칭 관계

**대표 예시:**

> 함수 $f(x) = a^{x-2} + 1$ ($a > 0$, $a \neq 1$)의 역함수 $g(x)$에 대하여
> $g(2)$를 구하시오.

**풀이 전략 및 사고 흐름:**

```
Step 1. 역함수 구하기
  → $y = a^{x-2} + 1$ → $x \leftrightarrow y$ 교환
  → $x = a^{y-2} + 1$ → $x - 1 = a^{y-2}$
  → $y - 2 = \log_a(x-1)$ → $y = \log_a(x-1) + 2$
  → $g(x) = \log_a(x-1) + 2$

Step 2. $g(2)$ 계산
  → $g(2) = \log_a(2-1) + 2 = \log_a 1 + 2 = 0 + 2 = 2$

※ 패턴 인식: $f(x)$의 점근선이 $y = 1$이면 $g(x)$의 점근선은 $x = 1$
```

---

#### 유형 3 | 그래프 + 정수 조건

**대표 예시:**

> $\log_2(x-3)$이 정의되도록 하는 모든 자연수 $x$에 대하여
> $\log_2(x-3)$이 정수가 되는 자연수 $x$의 개수를 $1 \leq x \leq 20$ 범위에서 구하시오.

**풀이 전략 및 사고 흐름:**

```
Step 1. 진수 조건
  → $x - 3 > 0$ → $x > 3$ → 자연수: $x \geq 4$

Step 2. $\log_2(x-3) = k$ (정수) 조건
  → $x - 3 = 2^k$ → $x = 2^k + 3$

Step 3. $4 \leq x \leq 20$ 범위에서 탐색
  → $k = 0$: $x = 4$ ✓
  → $k = 1$: $x = 5$ ✓
  → $k = 2$: $x = 7$ ✓
  → $k = 3$: $x = 11$ ✓
  → $k = 4$: $x = 19$ ✓
  → $k = 5$: $x = 35$ ✗ (범위 초과)
  → 총 5개
```

### Pattern 05 Part2 | 삼각함수 값 심화

#### 수능·평가원 핵심 유형 분류

| 세부 유형 | 대표 형태 | 전략 |
|---|---|---|
| 유형 1 | 변환 공식 연쇄 적용 | $\sin(-\theta)$, $\cos(\pi \pm \theta)$ 등 단계적 정리 |
| 유형 2 | 삼각함수 값으로 식 계산 | $\sin\theta$, $\cos\theta$ 구한 뒤 대입 |
| 유형 3 | 특수각 포함 조건 | $30°$, $45°$, $60°$ 등 정확한 값 활용 |

---

#### 유형 1 | 변환 공식 연쇄 적용

**대표 예시:**

> $\cos\!\left(\dfrac{\pi}{2} + \theta\right) = -\dfrac{1}{5}$일 때,
> $\dfrac{\sin\theta}{1 - \cos^2\theta}$의 값을 구하시오.

**풀이 전략 및 사고 흐름:**

```
Step 1. 변환 공식 적용
  → $\cos\!\left(\frac{\pi}{2} + \theta\right) = -\sin\theta = -\frac{1}{5}$
  → $\sin\theta = \frac{1}{5}$

Step 2. 분모 정리
  → $1 - \cos^2\theta = \sin^2\theta$

Step 3. 계산
  → $\frac{\sin\theta}{\sin^2\theta} = \frac{1}{\sin\theta} = \frac{1}{\frac{1}{5}} = 5$

※ 핵심: $1 - \cos^2\theta = \sin^2\theta$는 피타고라스 항등식의 변형
```

---

#### 유형 2 | 삼각함수 값으로 식 계산

**대표 예시:**

> $\dfrac{3}{2}\pi < \theta < 2\pi$이고 $\sin(-\theta) = \dfrac{1}{3}$일 때,
> $\tan\theta$의 값을 구하시오.

**풀이 전략 및 사고 흐름:**

```
Step 1. $\sin\theta$ 결정
  → $\sin(-\theta) = -\sin\theta = \frac{1}{3}$
  → $\sin\theta = -\frac{1}{3}$

Step 2. 사분면 판별
  → $\frac{3}{2}\pi < \theta < 2\pi$: 4사분면
  → $\cos\theta > 0$

Step 3. $\cos\theta$ 계산
  → $\cos\theta = \sqrt{1 - \sin^2\theta} = \sqrt{1 - \frac{1}{9}} = \frac{2\sqrt{2}}{3}$

Step 4. $\tan\theta$ 계산
  → $\tan\theta = \frac{\sin\theta}{\cos\theta} = \frac{-\frac{1}{3}}{\frac{2\sqrt{2}}{3}} = -\frac{1}{2\sqrt{2}} = -\frac{\sqrt{2}}{4}$
```

---

#### 변환 공식 완전 정리 (암기 필수)

$$\sin(\pi - \theta) = \sin\theta, \quad \cos(\pi - \theta) = -\cos\theta, \quad \tan(\pi - \theta) = -\tan\theta$$
$$\sin(\pi + \theta) = -\sin\theta, \quad \cos(\pi + \theta) = -\cos\theta, \quad \tan(\pi + \theta) = \tan\theta$$
$$\sin\!\left(\frac{\pi}{2} - \theta\right) = \cos\theta, \quad \cos\!\left(\frac{\pi}{2} - \theta\right) = \sin\theta$$
$$\sin\!\left(\frac{\pi}{2} + \theta\right) = \cos\theta, \quad \cos\!\left(\frac{\pi}{2} + \theta\right) = -\sin\theta$$

### Pattern 06 Part2 | 삼각함수 그래프 심화

#### 수능·평가원 핵심 유형 분류

| 세부 유형 | 대표 형태 | 전략 |
|---|---|---|
| 유형 1 | 그래프 조건으로 $a$, $b$, $c$, $d$ 결정 | 진폭·주기·최댓값 역산 |
| 유형 2 | 방정식 실근 개수 = 그래프 교점 수 | 직선 $y = k$와 교점 시각화 |
| 유형 3 | 최댓값·최솟값 조건 (닫힌구간) | 구간 내 극값·끝점 비교 |

---

#### 유형 1 | 그래프 조건 → 계수 역산

**대표 예시:**

> 함수 $f(x) = a\cos(bx) + 3$ ($a > 0$, $b > 0$)의 그래프에서
> 최댓값이 $13$이고 $x = \dfrac{\pi}{3}$에서 최댓값을 가질 때, $a + b$의 값을 구하시오.

**풀이 전략 및 사고 흐름:**

```
Step 1. 진폭에서 $a$ 결정
  → 최댓값 = $a + 3 = 13$ → $a = 10$

Step 2. 최댓값 위치에서 $b$ 결정
  → $\cos(bx)$의 최댓값은 $bx = 2k\pi$ ($k$는 정수)일 때
  → $x = \frac{\pi}{3}$에서 최댓값: $b \cdot \frac{\pi}{3} = 2k\pi$
  → $b = 6k$ → 최솟값 $b = 6$ ($k = 1$, $b > 0$)
  
  ※ $k=1$이 최솟값임을 확인:
    주기 = $\frac{2\pi}{b} = \frac{2\pi}{6} = \frac{\pi}{3}$

Step 3. $a + b = 10 + 6 = 16$
```

---

#### 유형 2 | 방정식 실근 개수 → 교점 수

**대표 예시:**

> $0 \leq x \leq 2\pi$에서 방정식 $2\sin x - \sqrt{3} = 0$의 실근의 개수를 구하시오.

**풀이 전략 및 사고 흐름:**

```
Step 1. 변환
  → $\sin x = \frac{\sqrt{3}}{2}$

Step 2. 그래프 교점 시각화
  → $y = \sin x$와 $y = \frac{\sqrt{3}}{2}$의 교점 수
  → $0 \leq x \leq 2\pi$에서 $\sin x = \frac{\sqrt{3}}{2}$:
     $x = \frac{\pi}{3}$, $x = \pi - \frac{\pi}{3} = \frac{2\pi}{3}$

→ 실근 2개
```

---

#### 유형 3 | 닫힌구간 최댓값·최솟값

**대표 예시:**

> 닫힌구간 $[0, 2\pi]$에서 $f(x) = a\cos bx + 3$이 $x = \dfrac{\pi}{3}$에서 최댓값 $13$을 갖도록 하는 두 자연수 $a$, $b$의 순서쌍 $(a, b)$에 대하여 $a + b$의 최솟값을 구하시오.

**풀이 전략 및 사고 흐름:**

```
Step 1. 최댓값 조건
  → $f\!\left(\frac{\pi}{3}\right) = a\cos\!\left(\frac{b\pi}{3}\right) + 3 = 13$
  → $a\cos\!\left(\frac{b\pi}{3}\right) = 10$

Step 2. $\cos\!\left(\frac{b\pi}{3}\right)$ 가능한 값 탐색
  → $b = 1$: $\cos\frac{\pi}{3} = \frac{1}{2}$ → $a = 20$, $a+b = 21$
  → $b = 2$: $\cos\frac{2\pi}{3} = -\frac{1}{2}$ → $a = -20$ (음수, 불가)
  → $b = 3$: $\cos\pi = -1$ → $a = -10$ (음수, 불가)
  → $b = 6$: $\cos 2\pi = 1$ → $a = 10$, $a+b = 16$
  → $b = 12$: $\cos 4\pi = 1$ → $a = 10$, $a+b = 22$

Step 3. 닫힌구간 조건 확인 후 최솟값 결정
  → $b = 6$일 때 $a + b = 16$ ← 최솟값 후보
```

### Pattern 07 Part2 | 사인·코사인 법칙 심화

#### 수능·평가원 핵심 유형 분류

| 세부 유형 | 대표 형태 | 전략 |
|---|---|---|
| 유형 1 | 두 법칙 연립 | 사인법칙으로 $R$ 구한 뒤 코사인법칙 연계 |
| 유형 2 | 넓이 + 법칙 혼합 | $S = \frac{1}{2}ab\sin C$와 법칙 연립 |
| 유형 3 | 삼각형 내 보조선·특수 구조 | 보조 삼각형 분리 후 각 법칙 적용 |

---

#### 유형 1 | 두 법칙 연립

**대표 예시:**

> 삼각형 $ABC$에서 $\overline{AB} = 3$, $\overline{BC} = \sqrt{13}$, $\overline{AD} \times \overline{CD} = 9$, $\angle BAC = \dfrac{\pi}{3}$일 때,
> $\sin\angle ADC$의 값을 구하시오. (단, $D$는 삼각형 $ABC$의 꼭짓점이 아님)

**풀이 전략 및 사고 흐름:**

```
Step 1. 주어진 변·각 정보 정리
  → $\angle BAC = \frac{\pi}{3}$, 두 변 $AB = 3$, $BC = \sqrt{13}$

Step 2. 코사인법칙으로 $AC$ 계산
  → $BC^2 = AB^2 + AC^2 - 2 \cdot AB \cdot AC \cdot \cos(\angle BAC)$
  → $13 = 9 + AC^2 - 2 \cdot 3 \cdot AC \cdot \frac{1}{2}$
  → $AC^2 - 3AC - 4 = 0$ → $(AC-4)(AC+1) = 0$
  → $AC = 4$ ($AC > 0$)

Step 3. $AD \times CD = 9$ 조건으로 사인법칙 연계
  → 삼각형 $ACD$의 외접원에서 사인법칙 적용
  → $\sin\angle ADC = \frac{AC}{2R}$ 활용
```

---

#### 유형 2 | 넓이와 법칙 혼합

**대표 예시:**

> 삼각형 $ABC$에서 $\sin A : \sin C = 8 : 5$이고,
> 삼각형 $ADE$와 삼각형 $ABC$의 넓이의 비가 $9 : 35$일 때,
> 삼각형 $ABC$의 외접원의 반지름의 길이가 $7$이면 삼각형 $PBC$의 넓이의 최댓값을 구하시오.

**풀이 전략 및 사고 흐름:**

```
Step 1. 사인법칙으로 변의 비 결정
  → $\frac{a}{\sin A} = \frac{c}{\sin C} = 2R$
  → $\sin A : \sin C = a : c = 8 : 5$

Step 2. 외접원 반지름으로 실제 변의 길이 결정
  → $2R = 14$ → $a = 2R\sin A$
  → $\sin A : \sin C = 8 : 5$에서 $a : c = 8 : 5$

Step 3. 넓이 비 조건 적용
  → 넓이 $= \frac{1}{2}ab\sin C$
  → 비 조건으로 미지의 변 결정

Step 4. 삼각형 $PBC$ 넓이 최댓값
  → $P$가 원 위의 점 → $BC$로부터 거리 최대일 때 넓이 최대
  → 원의 중심에서 $BC$까지의 거리 + 반지름으로 최대 높이 계산
```

---

#### 유형 3 | 두 삼각형 분리 후 각 법칙 적용

**사고 흐름 템플릿:**

```
복잡한 삼각형 문제 (사각형, 보조선 포함) 접근 순서:

1. 전체 도형을 삼각형 단위로 분리
2. 각 삼각형에서 알려진 변·각 목록 작성
3. 외접원 등장 → 사인법칙 적용 우선
4. 변 3개 또는 변 2개 + 끼인각 → 코사인법칙
5. 넓이 조건 → $S = \frac{1}{2}ab\sin C$ 연립
6. 구하는 값이 각도이면 최종적으로 역삼각함수 or 특수각 확인
```

### Pattern 10 Part2 | 점화식·수학적 귀납법 심화

#### 수능·평가원 핵심 유형 분류

| 세부 유형 | 대표 형태 | 전략 |
|---|---|---|
| 유형 1 | $a_{n+1} = pa_n + q$ | $b_n = a_n + c$ 치환 → 등비수열 |
| 유형 2 | $a_{n+1} = pa_n + f(n)$ | 양변 합산 → $a_n = a_1 + \sum f(k)$ |
| 유형 3 | 조건부 점화식 ($|a_n|$ 홀짝) | 직접 나열 + 역추적 |
| 유형 4 | 주기성 점화식 | $a_{n+k}$와 $a_n$ 관계로 주기 파악 |

---

#### 유형 1 | $pa_n + q$ 형 치환

**대표 예시:**

> 수열 $\{a_n\}$이 $a_1 = 1$, $a_{n+1} = 3a_n + 4$를 만족할 때, 일반항 $a_n$을 구하시오.

**풀이 전략 및 사고 흐름:**

```
Step 1. 치환값 결정
  → $a_{n+1} + c = 3(a_n + c)$ 형태로 만들기
  → $a_{n+1} + c = 3a_n + 3c$ → $3a_n + 4 + c = 3a_n + 3c$
  → $4 = 2c$ → $c = 2$

Step 2. $b_n = a_n + 2$로 치환
  → $b_{n+1} = 3b_n$ (등비수열, 공비 $3$)
  → $b_1 = a_1 + 2 = 3$
  → $b_n = 3 \cdot 3^{n-1} = 3^n$

Step 3. 역치환
  → $a_n = b_n - 2 = 3^n - 2$
```

---

#### 유형 2 | 합산 점화식

**대표 예시:**

> 수열 $\{a_n\}$이 $a_1 = 2$, $a_{n+1} = a_n + 2n + 1$을 만족할 때, $a_{10}$을 구하시오.

**풀이 전략 및 사고 흐름:**

```
Step 1. 점화식 변형
  → $a_{n+1} - a_n = 2n + 1$

Step 2. $n = 1$부터 $n = 9$까지 양변 합산
  → $a_{10} - a_1 = \sum_{n=1}^{9}(2n+1)$
  → $= 2 \cdot \frac{9 \cdot 10}{2} + 9 = 90 + 9 = 99$

Step 3. $a_{10}$ 계산
  → $a_{10} = a_1 + 99 = 2 + 99 = 101$
```

---

#### 유형 3 | 조건부 점화식 (수능 22번 유형)

**풀이 원칙 완전 정리:**

```
[조건부 점화식 표준 접근법]

① 규칙 분류
   - 각 조건별로 수열의 값이 어떻게 변하는지 명확히 정리
   - ex) 홀수 → 감소, 짝수 → 절반으로 감소, 0 → 고정

② 수렴 구조 파악
   - 대부분의 조건부 점화식은 결국 0 또는 특정 값으로 수렴
   - 수렴 후 구조 파악: $a_m = a_{m+2}$가 되는 시점 결정 가능

③ 목표 역추적
   - 조건 (나)처럼 "최솟값이 $m$" 형태 → $m-1$까지는 불만족, $m$에서 최초 만족
   - 역방향으로: "$m-2$와 $m$이 처음으로 같아지는" 시점 추적

④ 체계적 나열
   - 가능한 $a_1$의 범위를 홀짝·부호별로 분류
   - 각 범주에서 조건 (나) 만족 여부 확인
   - 조건 만족하는 $a_1$ 전부 수집 후 합산

[예시: 2025학년도 수능 22번 풀이 흐름]

점화식:
  홀수 → $a_{n+1} = a_n - 3$
  짝수·0 → $a_{n+1} = \frac{1}{2}a_n$

조건 (나): $|a_m| = |a_{m+2}|$인 최솟값 $m = 3$
→ $m=1$: $|a_1| \neq |a_3|$, $m=2$: $|a_2| \neq |a_4|$
→ $m=3$: $|a_3| = |a_5|$ (최초 성립)

분류별 추적:
  Case 1. $a_1 > 0$ 홀수
    → $a_2 = a_1 - 3$ (짝수 또는 홀수)
    → 이후 궤도 추적...

  Case 2. $a_1 > 0$ 짝수
    → $a_2 = \frac{a_1}{2}$
    → 이후 궤도 추적...

  Case 3. $a_1 < 0$ 홀수/짝수
    → $|a_1|$이 홀수: $a_2 = a_1 - 3$ (더 음수)
    → 이후 궤도 추적...

각 케이스에서 조건 (나) 만족 여부 확인 → 유효한 $a_1$ 수집
```

#### 유형 4 | 주기성 점화식

**대표 예시:**

> 수열 $\{a_n\}$이 $a_n + a_{n+4} = 12$를 만족시키고
> $a_1 = 1$, $a_2 = 2$, $a_3 = 3$, $a_4 = 4$일 때,
> $\displaystyle\sum_{n=1}^{16}a_n$을 구하시오.

**풀이 전략 및 사고 흐름:**

```
Step 1. 주기 파악
  → $a_{n+4} = 12 - a_n$
  → $a_{n+8} = 12 - a_{n+4} = 12 - (12-a_n) = a_n$ → 주기 8

Step 2. 한 주기 합
  → $(a_1 + a_5) + (a_2 + a_6) + (a_3 + a_7) + (a_4 + a_8) = 4 \times 12 = 48$

Step 3. $n=1$~$16$까지 합
  → $\sum_{n=1}^{16} = 2 \times 48 = 96$
```

```
[21번 유형 - 함수 조건 복합]

1. 점화식 규칙을 명확히 정리 (조건별 표 작성)
2. 수렴/순환 구조 파악 (직접 나열)
3. 조건 (나)의 최솟값 조건 → 역추적으로 경우 분류
```
