## 1장 함수의 극한과 연속

---

### Pattern 01 | 좌극한, 우극한, 극한값, 함숫값을 구분하라!

**핵심 원리:**
- $x \to a$는 "$x \neq a$"이면서 $a$에 한없이 가까워진다는 의미 (절대 $a$가 될 수 없음)
- **극한값 ≠ 함숫값**: $\lim_{x \to a}f(x) \neq f(a)$일 수 있다
- **좌극한**: $x < a$이면서 $a$로 가까워질 때의 $y$값
- **우극한**: $x > a$이면서 $a$로 가까워질 때의 $y$값
- **극한값 존재 조건**: 좌극한 = 우극한

**치환 기술 (자주 출제):**
- $x \to \infty$일 때 $f\!\left(\frac{1}{x}\right)$의 극한 → $\frac{1}{x} = t$로 치환하면 $t \to 0^+$
- $f(-x) = -f(x)$ (기함수) 조건: $\lim_{x \to a^-}f(x) = -\lim_{t \to (-a)^+}f(t)$로 변환 가능 → 좌극한을 우극한으로 변환

**그래프 읽기 3단계 (반드시 이 순서로):**
1. 좌극한 확인 ($x$가 왼쪽에서 $a$로 접근할 때 $y$값)
2. 우극한 확인 ($x$가 오른쪽에서 $a$로 접근할 때 $y$값)
3. 함숫값 확인 (채워진 점 vs 빈 점)

**AI 튜터 핵심 발문:**
- "지금 극한값을 구하는 건가요, 함숫값을 구하는 건가요? 그 둘이 같은가요, 다른가요?"
- "좌극한과 우극한을 각각 구해보셨나요? 두 값이 같아야 극한값이 존재해요."
- "$x \to a^-$일 때와 $x \to a^+$일 때 그래프에서 $y$값이 각각 어디로 가까워지나요?"

**학생 오류 패턴:**
- 극한값과 함숫값을 혼동하는 것 → "극한값은 $f(a)$가 아니에요. $x$가 $a$에 가까워질 때의 값이에요."
- 좌극한만 확인하고 극한값이 존재한다고 판단하는 것
- 그래프에서 열린 점(빈 점)을 함숫값으로 읽는 것

### Pattern 02 | 모두 수렴하는 함수로 표현하여 극한값을 구하라!

**핵심 원리: 극한 계산의 핵심은 '수렴하는 함수로 표현하기'**

극한의 성질로 $\lim$을 분배하려면 **모든 항이 수렴하는 함수**여야 한다.

**상황별 수렴 변환 전략 4가지:**

**① $\frac{\infty}{\infty}$ 꼴 (다항함수, $x \to \infty$):**
- 분모·분자를 **최고차항**으로 나눠 수렴 변환
- 판정 규칙 (암기):
  - $g(x)$의 차수 > $f(x)$의 차수 → 발산
  - $g(x)$의 차수 < $f(x)$의 차수 → 0으로 수렴
  - 차수 같음 → **최고차항 계수의 비**로 수렴

**② 루트 발산 꼴 ($\sqrt{} - \sqrt{}$ 또는 $\sqrt{} +$ 다항식):**
- 분모·분자에 **켤레식**을 곱하여 유리화 → 수렴하는 함수로 변환

**③ $\frac{0}{0}$ 꼴 ($x \to a$):**
- 분모가 0이 되지 않도록 **인수분해 후 약분**

**④ '미정계수의 결정' — 수능 고난도 핵심 전략:**

> **미정계수의 결정 1:** $\lim_{x \to a}\frac{f(x)}{g(x)}$의 극한값이 존재할 때,
> $\lim_{x \to a} g(x) = 0$이면 $\lim_{x \to a} f(x) = 0$이다.

> **미정계수의 결정 2:** $\lim_{x \to a}\frac{f(x)}{g(x)}$의 극한값이 존재하는데 그 값이 $0$이 아닐 때,
> $\lim_{x \to a} f(x) = 0$이면 $\lim_{x \to a} g(x) = 0$이다.

→ **요약:** "수렴하는 극한에서 분모·분자 중 하나가 0으로 가면 나머지도 0으로 간다"

**미정계수 결정 활용 절차:**
1. 분모가 0으로 수렴하는 모든 점에서 분자도 0으로 수렴해야 한다
2. 이를 이용해 함수의 인수 또는 계수 결정
3. $\{f(x)=0\text{의 모든 실근}\} \subset \{f(2x+1)=0\text{의 모든 실근}\}$ 형태로도 활용 가능

**AI 튜터 핵심 발문:**
- "지금 이 극한이 $\frac{\infty}{\infty}$ 꼴인가요, $\frac{0}{0}$ 꼴인가요? 각각 어떻게 변환해야 할까요?"
- "분모가 0으로 수렴하네요. 그렇다면 분자도 반드시 0으로 수렴해야 해요. 그 조건에서 무엇을 알 수 있나요?"
- "극한값이 존재하는데 분모가 0이면, 분자에 어떤 인수가 있어야 할까요?"
- "루트가 있다면 켤레식을 곱해서 유리화해보셨나요?"

**학생 오류 패턴:**
- 수렴하지 않는 항이 있는데 $\lim$을 분배하는 것
- 미정계수 결정 조건을 쓰지 않고 무작정 대입하는 것
- 루트 발산 꼴에서 켤레식 대신 단순 대입을 시도하는 것

### Pattern 03 | '불연속 후보 찾기·구간별 함수 쓰기'를 익혀라!

**핵심 원리 (3가지 실전 개념 암기):**

**① 사칙연산 함수의 불연속 후보 찾기:**
- $f(x)$, $g(x)$가 사칙연산으로 이루어진 함수의 불연속 후보는:
  - $f(x)$가 불연속인 $x$
  - $g(x)$가 불연속인 $x$
  - (나누기라면) 분모 $= 0$이 되는 $x$도 추가

**② 곱함수의 연속성 (핵심 치트키):**
- $x = a$에서 연속인 $f(x)$ + 좌극한·우극한·함숫값은 존재하지만 불연속인 $g(x)$에 대하여:
  - **$f(a) = 0$이면 $f(x)g(x)$는 $x = a$에서 연속**
  - $f(a) \neq 0$이면 $f(x)g(x)$는 $x = a$에서 불연속

**③ 구간별 함수의 연속성:**
$$p(x) = \begin{cases} f(x) & (x < a) \\ g(x) & (x \geq a) \end{cases} \text{가 } x=a\text{에서 연속} \Leftrightarrow f(a) = g(a)$$

**연속성 문제의 2가지 해법:**
1. **불연속 후보 찾기 → 연속의 정의로 확인** (좌극한 = 우극한 = 함숫값)
2. **구간별 함수 쓰기 → 경계점 대입하기**

**고난도 유형 전략:**
- $f(x-a)$의 불연속: $f(x)$의 불연속점이 $x=b$이면 $f(x-a)$는 $x=a+b$에서 불연속 (평행이동)
- $(x-a)f(x)$ 꼴: $f(x)$가 $x=a$에서 불연속이어도 $(x-a)=0$이므로 곱함수 연속성 적용 가능
- $\{f(x)\}^2$ 꼴: 제곱 후 불연속 후보 재확인 필요 ($f(x)$가 불연속이어도 $\{f(x)\}^2$은 연속일 수 있음)
- $f(x)f(x-a)$ 꼴: $f(x)$의 불연속점과 $f(x-a)$의 불연속점 모두 확인
- $h(x) = f(x-3) \times \lim_{t \to 0^+}\frac{|f(x+t)|-|f(x-t)|}{t}$ 꼴: $f'(x)$의 부호 변화 지점도 불연속 후보

**AI 튜터 핵심 발문:**
- "이 함수의 불연속 후보가 어디인가요? 사칙연산 구조를 보면 어디서 불연속이 생길 수 있나요?"
- "불연속 후보를 찾았다면, 그 점에서 좌극한, 우극한, 함숫값을 각각 구해서 연속 정의와 비교해보세요."
- "$f(a) = 0$인 점이 있나요? 그 점에서 곱함수는 연속이 될 수 있어요."
- "구간별로 주어진 함수라면 경계점에서 두 함수의 값이 같은지 바로 대입해보세요."

**학생 오류 패턴:**
- 불연속 후보를 찾고 나서 연속 정의로 확인하지 않고 결론 내리는 것
- 곱함수에서 $f(a) = 0$인 경우 연속이 된다는 것을 모르는 것
- 평행이동된 함수 $f(x-a)$의 불연속점을 $f(x)$의 불연속점과 같다고 착각하는 것
- $\{f(x)\}^2$ 꼴에서 $f(x)$의 불연속점이 $\{f(x)\}^2$의 불연속점이 아닐 수도 있음을 모르는 것

### Pattern 04 | 좌미분계수, 우미분계수가 같아야 미분가능하다

**핵심 원리:**

**미분계수의 정의 (반드시 암기):**
$$f'(a) = \lim_{h \to 0} \frac{f(a+h)-f(a)}{h} = \lim_{b \to a} \frac{f(b)-f(a)}{b-a}$$

**미분가능 조건 (동치 관계 — 3가지 표현 모두 암기):**
$$\text{미분계수가 존재한다}$$
$$\Leftrightarrow f'(a)\text{의 값이 존재한다}$$
$$\Leftrightarrow \lim_{h \to 0^+}\frac{f(a+h)-f(a)}{h} = \lim_{h \to 0^-}\frac{f(a+h)-f(a)}{h}$$
→ **좌미분계수 = 우미분계수**

**미분계수 계산 2가지 핵심 기술:**

**① 강제로 나누고 곱하기 (가장 중요! 강제성이 핵심):**
$$\lim_{h \to 0}\frac{f(1+3h)-f(1)}{h} = \lim_{h \to 0}\frac{f(1+3h)-f(1)}{3h} \times 3 = 3f'(1)$$
- $f(\star)-f(\bigcirc)$ 모양이 보이면 → $\star - \bigcirc$을 **강제로 나누고 곱해서** 기울기 모양으로 만들기
- **분모를 신경 쓰지 말고 분자에서 차이($\star - \bigcirc$)를 찾아라**

**② 분자를 함수로 치환하기:**
$$\lim_{x \to 2}\frac{f(x)-8x+1}{x-2} \xrightarrow{g(x)=f(x)-8x+1} \lim_{x \to 2}\frac{g(x)-g(2)}{x-2} = g'(2) = f'(2)-8$$
- 분자가 복잡하면 통째로 $g(x)$로 치환 → $g(a) = 0$ 확인 → 미분계수 정의 적용

**미분계수의 기하적 의미:** $f'(a)$ = 점 $(a, f(a))$에서의 접선의 기울기

**AI 튜터 핵심 발문:**
- "$f(\star)-f(\bigcirc)$ 모양이 보이죠? $\star - \bigcirc$이 무엇인가요? 그것으로 강제로 나누고 곱해보세요."
- "분모가 $h$이고 분자에 $f(1+3h)-f(1)$이 있다면, $3h$로 강제로 나누고 $3$을 곱해보세요."
- "분자가 복잡하다면 $g(x) =$분자 로 치환하면 $g(a) = 0$이 되나요? 그렇다면 미분계수 정의로 바꿀 수 있어요."
- "좌미분계수와 우미분계수를 각각 구했나요? 둘이 같아야 미분가능해요."

**학생 오류 패턴:**
- $\lim_{h \to 0}\frac{f(a+3h)-f(a)}{h}$에서 $3$을 놓치는 것 (강제로 나누고 곱하지 않음)
- 미분가능성 확인 시 연속성만 확인하는 것 (연속 ≠ 미분가능!)
- 분자 치환 시 $g(a) = 0$임을 확인하지 않는 것

### Pattern 05 | '미분불가능 후보 찾기·구간별 함수 쓰기'를 숙지하라!

**핵심 원리 (Pattern 03 연속성과 구조 동일):**

**구간별 함수의 미분가능성:**
$$p(x) = \begin{cases} f(x) & (x < a) \\ g(x) & (x \geq a) \end{cases} \text{가 } x=a\text{에서 미분가능}$$
$$\Leftrightarrow \underbrace{f(a) = g(a)}_{\text{연속}} \quad \text{AND} \quad \underbrace{f'(a) = g'(a)}_{\text{좌미분 = 우미분}}$$

→ **연속이면서 미분가능 두 조건 동시 만족 필요**

**사칙연산 함수의 미분불가능 후보:**
- $f(x)$가 미분불가능인 $x$
- $g(x)$가 미분불가능인 $x$
- 나누기라면 분모 $= 0$이 되는 점도 추가

**곱함수의 미분가능성 (핵심 치트키):**
- $x = a$에서 미분가능한 $f(x)$ + $x = a$에서 연속이지만 미분불가능한 $g(x)$에 대하여:
  - **$f(a) = 0$이면 $f(x)g(x)$는 $x = a$에서 미분가능**

**미분불가능 후보 찾기 → 미분계수 정의로 확인** 2단계 절차

**AI 튜터 핵심 발문:**
- "미분불가능 후보가 어디인가요? 구간별로 끊기는 점, 절댓값 $= 0$인 점, 분모가 0인 점을 모두 확인해보세요."
- "경계점에서 ① 연속인지, ② 좌미분계수와 우미분계수가 같은지 두 가지 모두 확인해야 해요."
- "$f(a) = 0$인 경우 곱함수가 미분가능할 수 있어요. 정의로 직접 구해보세요."

**학생 오류 패턴:**
- 연속만 확인하고 미분가능하다고 결론 내리는 것 (연속 ≠ 미분가능!)
- 미분불가능 후보를 찾고 나서 정의로 확인하지 않는 것
- $|f(x)|$ 꼴에서 $f(x) = 0$인 점의 미분가능성을 확인하지 않는 것

### Pattern 06 | 접선 문제는 $y = f'(t)(x-t) + f(t)$를 쓰고 시작하라!

**핵심 원리:**
- 접선 문제는 유형에 관계없이 **무조건 $y = f'(t)(x-t) + f(t)$부터 쓰자**

**접선의 3가지 유형별 풀이:**

**유형 ①: 곡선 위의 점 $(t, f(t))$이 주어진 경우**
- $t$와 $f(t)$ 모두 주어짐 → 바로 대입하여 접선 완성

**유형 ②: 접선의 기울기 $k$가 주어진 경우**
- $f'(t) = k$를 풀어서 $t$ 결정 → 대입

**유형 ③: 곡선 밖의 점 $(a, b)$를 지나는 접선**
- $y = f'(t)(x-t) + f(t)$에 $x = a$, $y = b$ 대입
- $t$에 대한 방정식 풀기 → 접선 완성

**AI 튜터 핵심 발문:**
- "접선 문제네요. 무조건 $y = f'(t)(x-t) + f(t)$부터 써보세요."
- "주어진 것이 곡선 위의 점인가요, 기울기인가요, 곡선 밖의 점인가요?"
- "곡선 밖의 점 $(a,b)$가 주어졌다면, 그 점을 접선 방정식에 대입해서 $t$에 대한 방정식을 세워보세요."

**학생 오류 패턴:**
- 접선 공식을 외우지 않고 문제마다 유도하는 것
- 유형 ③에서 곡선 밖의 점을 접선 방정식에 대입하지 않고 헤매는 것
- 기울기가 주어진 유형 ②에서 $f'(t) = k$ 방정식을 세우지 않는 것

### Pattern 07 | 극값의 정의·판정을 암기하라. 도함수의 부호가 그래프의 핵심이다

**핵심 원리 (3가지 주제):**

**① 극값의 판정 (조건반사처럼 적용):**
$$f(x)\text{가 } x=a\text{에서 미분가능하고 극값을 가지면} \Rightarrow f'(a) = 0$$
→ **미분 문제에서 부등식이 보이면 항상 '극값의 정의 → 극값의 판정' 활용 고려**

**② 극값은 부등식으로 정의한다:**
- $x = a$를 포함하는 어떤 열린구간에 속하는 모든 $x$에 대해 $f(x) \geq f(a)$이면 $f(a)$는 극솟값
- 부등식 조건이 주어지면 → 극값의 정의를 통해 $f'(a) = 0$ 도출

**③ 도함수의 부호 = 그래프의 핵심 (반드시 암기):**
- $f'(x) > 0$인 구간 → $f(x)$ 증가
- $f'(x) < 0$인 구간 → $f(x)$ 감소
- $f'(a) = 0$이고 부호가 $+ \to -$ 변화 → 극대
- $f'(a) = 0$이고 부호가 $- \to +$ 변화 → 극소
- $f'(a) = 0$이지만 부호 변화 없음 → **극값 아님** (변곡점 등)

**자주 출제되는 삼차·사차함수 그래프 형태:**

| 함수 | 극값 개수 | 그래프 특징 |
|---|---|---|
| 삼차함수 (최고차항 계수 양수, 극값 있음) | 극대·극소 각 1개 | $\cap\cup$ 형태 |
| 삼차함수 (극값 없음, $D \leq 0$) | 0개 | 단조증가 |
| 사차함수 (선대칭) | 극값 3개 | W 또는 M 형태 |
| 사차함수 (극값 1개) | 극솟값 1개 | U 형태 |

**닫힌구간 $[a, b]$에서 최댓값·최솟값 구하기:**
- **반드시 3곳에서 비교:** ① 극댓값, ② 극솟값, ③ 양 끝점 $f(a)$, $f(b)$

**수능 킬러 전략 (조건 추론형):**
- $f(-x) = -f(x)$ → 기함수 → 원점 대칭 → 극대·극솟값이 절댓값 같고 부호 반대
- 방정식 $f(x) = k$의 실근 개수 → 그래프와 직선 $y = k$의 교점 수로 해석 → 극값 조건 역산
- 역함수가 존재하려면 → 실수 전체에서 단조증가 or 단조감소 → $f'(x) \geq 0$ (판별식 $D \leq 0$)
- $g(x) = f(x-3) \times |f'(x)|$: $f'(x) = 0$이 되는 점, $f'(x)$ 부호 바뀌는 점 모두 불연속·미분불가능 후보

**AI 튜터 핵심 발문:**
- "도함수 $f'(x)$의 부호를 구간별로 분석해보셨나요? 부호가 바뀌는 점이 극값이에요."
- "$f'(a) = 0$이어도 극값이 아닐 수 있어요. 그 점에서 부호가 바뀌는지 확인해야 해요."
- "닫힌구간에서 최댓값·최솟값은 극값과 양 끝점에서만 나와요. 세 곳 모두 비교하셨나요?"
- "방정식 $f(x) = k$의 실근 개수를 구하려면 그래프와 직선 $y = k$의 교점으로 해석해보세요."
- "함수가 기함수인가요? 그렇다면 극대·극솟값 관계를 바로 알 수 있어요."

**학생 오류 패턴:**
- $f'(a) = 0$이면 무조건 극값이라고 생각하는 것 (부호 변화 확인 필수)
- 도함수 부호 분석 없이 그래프를 그리려는 것
- 삼차함수의 극값이 없는 경우를 놓치는 것 (판별식 $D \leq 0$ 확인)
- 닫힌구간에서 양 끝점 값을 비교하지 않는 것

### Pattern 08 | 속도, 가속도의 개념을 숙지하라!

**핵심 원리:**

수직선 위를 움직이는 점 P의 시각 $t$일 때의 위치 $x = f(t)$에 대하여:
$$v = \frac{dx}{dt} = f'(t) \quad (\text{속도}) \qquad a = \frac{dv}{dt} = \{f'(t)\}' \quad (\text{가속도})$$

**핵심 관계 (반드시 암기):**
- 속도 = 위치의 **도함수**
- 가속도 = 속도의 **도함수** = 위치의 **이계도함수**
- $f'(t)$의 부호 = **운동 방향** ($+$: 양의 방향, $-$: 음의 방향)
- **운동 방향이 바뀌는 순간** = 속도 $v = 0$이 되는 시각

**AI 튜터 핵심 발문:**
- "속도를 구하려면 위치 함수를 미분해야 해요. 가속도는 속도를 한 번 더 미분하면 돼요."
- "운동 방향이 바뀌는 순간은 속도가 $0$이 되는 시각이에요. 언제인가요?"
- "속도의 부호를 구간별로 분석하면 운동 방향을 알 수 있어요."

**학생 오류 패턴:**
- 평균속도(평균변화율)와 순간속도(미분계수)를 혼동하는 것
- 운동 방향이 바뀌는 조건을 가속도 $= 0$으로 잘못 이해하는 것 (속도 $= 0$이 정답)

### Pattern 09 | 부정적분 고난도 문항은 주로 차수추적이다!

**핵심 원리:**
- 부정적분 문제가 어려워지면 대부분 **차수추적**이 필요
- 주어진 식에서 다항함수의 **차수**를 먼저 알아내야 한다

**차수추적 전략:**
- $f(x) = \int xg(x)dx$ → $g(x)$가 $n$차이면 $f(x)$는 $n+2$차
- $\frac{d}{dx}\{f(x)-g(x)\} = 4x^3+2x$ → $f(x)-g(x)$는 4차 → $f(x)$, $g(x)$ 각각의 차수 추정 가능
- 도출된 차수로 $g(x) = ax^2+bx+c$ 등으로 놓고 계수 비교

**부정적분 문제가 어려우면 차수추적부터 생각하자!**

**AI 튜터 핵심 발문:**
- "이 식에서 $f(x)$와 $g(x)$의 차수를 먼저 파악해볼까요? 적분하면 차수가 몇 씩 올라가나요?"
- "차수를 알았다면 계수를 $a, b, c$로 놓고 주어진 조건에 대입해보세요."

**학생 오류 패턴:**
- 차수를 파악하지 않고 무작정 계수 비교를 시도하는 것
- 부정적분할 때 적분상수 $C$를 빠뜨리는 것

### Pattern 10 | 정적분의 속산, 기함수·우함수의 정적분을 숙지하라!

**핵심 원리:**

**① 정적분의 속산 공식 (암기):**
$$\int_0^1 x^n dx = \frac{1}{n+1}$$
→ **지수에 1 더해서 역수, 상수는 그대로** (암산 가능)

예: $\int_0^1(5x^4+4x^3+6x^2-1)dx = 5\cdot\frac{1}{5}+4\cdot\frac{1}{4}+6\cdot\frac{1}{3}-1 = 3$

**② 기함수·우함수의 정적분 (핵심 치트키):**

| 조건 | 공식 | 기억법 |
|---|---|---|
| $f(-x) = -f(x)$ (기함수, 홀수차항만) | $\int_{-c}^{c}f(x)dx = 0$ | 원점 대칭 → 넓이 상쇄 |
| $f(-x) = f(x)$ (우함수, 짝수차항+상수) | $\int_{-c}^{c}f(x)dx = 2\int_0^c f(x)dx$ | $y$축 대칭 → 두 배 |

**적분 구간이 $[-c, c]$를 보면 → 기함수·우함수 분리를 떠올려라**

예: $\int_{-3}^{3}(3x^7-2x^5+5x^4-2x^3+x+1)dx = 2\int_0^3(5x^4+1)dx$ (홀수차 항 소거)

**함수의 홀짝 판별:**
- $y = x^2+2x$처럼 짝수차항+홀수차항 혼합 → 기함수도 우함수도 아님
- $y = x^6-3x^4+2$처럼 짝수차항 및 상수항만 → **우함수**
- $h(x) = f(x)g(x)$에서 $f$가 기함수, $g$가 우함수 → $h$는 기함수

**AI 튜터 핵심 발문:**
- "적분 구간이 $[-c, c]$ 형태인가요? 그러면 기함수·우함수를 분리해서 계산이 훨씬 쉬워져요."
- "$f(-x) = -f(x)$이면 기함수, $f(-x) = f(x)$이면 우함수예요. 주어진 함수가 어디에 해당하나요?"
- "$\int_0^1 x^n dx$의 값을 바로 말할 수 있나요? $\frac{1}{n+1}$이에요."

**학생 오류 패턴:**
- 적분 구간이 $[-c, c]$인데 기함수·우함수 성질을 활용하지 않고 전부 계산하는 것
- 기함수·우함수 여부를 확인하지 않고 임의로 분리하는 것
- 홀수차항과 짝수차항 분리 후 홀수차항이 0이 됨을 인식하지 못하는 것

### Pattern 11 | 정적분으로 표현된 함수는 대입·미분이 핵심이다!

**핵심 원리 (저자 LECTURE 4가지 규칙):**

**① $\int_a^b f(t)dt$ 꼴 (양 끝이 상수):** 전부 **상수** → 숫자처럼 다루기

**② $\int_a^x f(t)dt$ 꼴 ($x$가 위끝):** $x$에 대한 **함수** → 미분 가능
$$\left(\int_a^x f(t)dt\right)' = f(x)$$

**③ $\int_a^x f(t)dt$를 보자마자 2가지를 동시 시행 (핵심!):**
- **미분하기**: 양변을 $x$에 대해 미분
- **$x = a$ 대입하기**: $\int_a^a f(t)dt = 0$ 이용하여 상수 항 결정

**④ $\int_a^x \star\, dt$에서 $\star$에 $t$ 이외의 변수(예: $x$)가 있으면:**
- $x$는 $t$와 무관하므로 적분 기호 밖으로 빼내기
- 예: $\int_0^x (x-t)f(t)dt = x\int_0^x f(t)dt - \int_0^x tf(t)dt$로 분리 후 미분

**$\lim_{x \to a}\frac{\int_a^x f(t)dt}{x-a} = f(a)$ (미분계수 정의와 연결)**

**AI 튜터 핵심 발문:**
- "$\int_a^x f(t)dt$가 보이네요. 바로 ① 양변 미분, ② $x = a$ 대입 두 가지를 동시에 해보세요."
- "$\int_3^1 f(t)dt$ 같은 경우는 완전히 상수예요. 숫자처럼 취급하세요."
- "피적분함수에 $x$와 $t$가 섞여 있나요? $x$를 적분 밖으로 꺼낼 수 있는지 확인해보세요."
- "$\lim_{x \to a}\frac{\int_a^x f(t)dt}{x-a}$의 형태가 보이면 바로 $f(a)$로 바꿀 수 있어요."

**학생 오류 패턴:**
- $\int_a^b f(t)dt$를 함수로 착각하는 것 (상수임!)
- 미분만 하고 $x = a$ 대입을 잊는 것 → 상수 항 결정 실패
- $\int_0^x(x-t)f(t)dt$에서 $x$를 분리하지 않고 미분하려는 것

### Pattern 12 | 정적분의 기하적 의미를 파악하되, 넓이는 그래프 없이 구하라!

**핵심 원리:**

**정적분의 기하적 의미:**
$$\int_a^b f(x)dx = \text{(곡선과 }x\text{축 사이 넓이)} \quad \text{단, }f(x) \geq 0\text{인 경우}$$

**넓이 계산 핵심 원칙: 그래프를 그릴 필요 없다!**
$$S = \int_a^b |f(x)|dx = \sum \left|\int_{\alpha_k}^{\alpha_{k+1}} f(x)dx\right|$$

**넓이 계산 절차:**
1. **방정식의 근 찾기**: $f(x) = 0$ 또는 $f(x) = g(x)$의 근
2. **구간별 정적분 계산**
3. **절댓값 취하기**: 각 구간 정적분의 절댓값을 더함

**두 곡선 사이 넓이:**
$$S = \int_a^b |f(x)-g(x)|dx = \sum \left|\int_{\alpha_k}^{\alpha_{k+1}} \{f(x)-g(x)\}dx\right|$$

**평행이동과 정적분:**
$$\int_a^b f(x)dx = \int_{a+k}^{b+k} f(x-k)dx$$
→ 평행이동해도 정적분 값은 **불변**

**영역 분할 문제 (고난도):**
- 전체 넓이 = $A + B + C$ 조건이 주어질 때
- 각 영역을 정적분으로 표현하고 조건으로 방정식 수립
- **핵심:** $(A의 넓이) + (C의 넓이) = (B의 넓이)$ → 정적분으로 표현 후 미지수 결정

**AI 튜터 핵심 발문:**
- "넓이 문제네요. 그래프를 그리지 않아도 돼요. 먼저 $f(x) = 0$의 근을 구해서 부호가 바뀌는 구간을 찾아보세요."
- "구간별로 정적분을 계산한 후, 각각 절댓값을 취해서 더하면 넓이가 나와요."
- "두 곡선 사이 넓이라면 $f(x) - g(x)$의 부호가 바뀌는 점을 먼저 찾아보세요."
- "평행이동을 해도 정적분 값은 변하지 않아요. 이 성질을 이용할 수 있는 부분이 있나요?"
- "두 영역의 넓이 차이를 하나의 정적분으로 합칠 수 있을까요?"

**학생 오류 패턴:**
- 그래프가 $x$축 아래인 구간에서 정적분 값이 음수임을 무시하고 넓이로 사용하는 것
- 넓이를 구할 때 절댓값 처리 없이 정적분을 그대로 더하는 것
- 영역 분할 문제에서 전체 넓이와 부분 넓이의 관계를 정적분으로 표현하지 못하는 것

### Pattern 13 | 속도, 위치의 변화량, 움직인 거리를 구분하라!

**핵심 원리 (3가지 공식 완벽 구분):**

수직선 위를 움직이는 점 P의 속도 $v(t) = f'(t)$, 시각 $t = a$일 때 위치 $f(a)$:

| 구분 | 공식 | 핵심 |
|---|---|---|
| **시각 $t$일 때 위치** | $f(t) = f(a) + \int_a^t v(t)dt$ | 초기 위치 + 변화량 |
| **위치의 변화량** | $\int_a^b v(t)dt$ | 부호 있음 (음수 가능) |
| **움직인 거리** | $\int_a^b |v(t)|dt$ | 항상 양수 (절댓값!) |

**결정적 차이:**
- **위치의 변화량** = 정적분 $\int_a^b v(t)dt$ (속도가 음수인 구간 상쇄됨, 정확한 위치가 아님!)
- **움직인 거리** = $\int_a^b |v(t)|dt$ (속도가 음수인 구간도 양수로 더함)

**계산 절차:**
1. 속도 $v(t)$에서 $v(t) = 0$이 되는 $t$ 찾기 (운동 방향 바뀌는 시각)
2. 구간별로 $v(t)$의 부호 결정
3. 움직인 거리: 구간별 정적분 후 절댓값 취해서 합산

**AI 튜터 핵심 발문:**
- "지금 구하는 게 위치의 변화량인가요, 움직인 거리인가요? 이 둘은 달라요."
- "속도가 $0$이 되는 시각이 언제인가요? 그 시각을 기준으로 구간을 나눠야 해요."
- "움직인 거리는 속도에 절댓값을 붙여서 적분해야 해요. 그냥 속도를 적분하면 변화량이에요."

**학생 오류 패턴:**
- 위치의 변화량과 움직인 거리를 같다고 생각하는 것 (가장 빈번)
- 속도가 음수인 구간을 고려하지 않고 그냥 정적분하는 것
- 운동 방향이 바뀌는 순간($v=0$)을 찾지 않는 것

| **[2단계] 핵심 조건 확인** | 미정계수 결정, 연속·미분가능 조건, 극값 판정, 정적분 vs 부정적분 구분 등 |
| **[3단계] 의사결정 질문** | 각 패턴의 핵심 공식·변환 전략으로 유도 |
| **[4단계] 모순 검증 유도** | "구한 값이 처음 조건(연속, 미분가능 등)을 만족하나요?" |
| **[절대 원칙]** | 어떤 경우에도 최종 정답이나 전체 풀이를 한 번에 제공하지 않는다 |

## 📎 수학Ⅰ ↔ 수학Ⅱ 공통 인지 구조 (범용성 근거)

| 수학Ⅰ (수열 킬러) | 수학Ⅱ (미분·적분 킬러) | 공통 인지 구조 |
|---|---|---|
| 홀수/짝수 조건으로 점화식 분기 | 극대·극소 조건으로 함수 분기 | **케이스 분류 → 모순 발견** |
| 자연수·정수 조건 역추적 | 연속·미분가능 조건 역추적 | **제약 조건 역추적** |
| 수형도로 케이스 시각화 | 도함수 부호표로 케이스 시각화 | **시각적 모델링** |
| 밑 조건·진수 조건 체크 | 좌극한·우극한 동시 확인 | **제약 조건 검증** |
| $\sum$의 4종류 판단 트리 | 정적분 계산법 판단 트리 | **의사결정 트리** |
| 미정계수를 조건에서 역산 | 미정계수 결정 (분모·분자 동시 0) | **미정계수 추적** |

**결론:** 수학Ⅰ의 10가지 패턴과 수학Ⅱ의 13가지 패턴은 표면적 주제만 다를 뿐,
**"조건 분류 → 케이스 탐색 → 모순 제거 → 검증"**이라는 M-CoT 4단계 구조는 완전히 동일하게 작동한다.
