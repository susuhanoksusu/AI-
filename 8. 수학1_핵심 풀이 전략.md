## 1장 지수함수와 로그함수

### Pattern 01 | 지수·로그는 모양을 위한 연산 선택, 유리수 지수로 통일이다

**핵심 원리 (2가지 서브 전략):**

**① 목표 식의 모양을 먼저 보고 연산을 선택한다**
- 구하려는 식이 $\frac{B}{A}$ → 주어진 로그식에서 **빼기**를 생각 ($\log_a B - \log_a A$)
- 구하려는 식이 $AB$ → 주어진 로그식에서 **더하기**를 생각
- 구하려는 식이 $a^{A-B}$ → 주어진 지수식에 **$(a-b)$ 제곱**을 취해서 등장시키기
- **핵심:** 무작정 계산부터 시작하지 말고, **"어떤 연산을 해야 목표 모양이 나오는가?"를 먼저 생각**

**② 모든 지수를 유리수 지수로 통일한다**
- $a^{\frac{m}{n}} = \sqrt[n]{a^m}$ 공식으로 $n$제곱근 기호를 유리수 지수로 변환
- 특히 $a^{\frac{1}{n}} = \sqrt[n]{a}$ → 이를 역으로 $\sqrt[n]{a} = a^{\frac{1}{n}}$으로 바꾸면 지수법칙 적용 가능
- 모든 $n$제곱근을 유리수 지수로 고치면 문제가 풀리는 경우가 많다

**로그 변환 치트키 2가지 (반드시 암기):**
- 상수 → 지수/로그로: $3 = 2^{\log_2 3} = \log_2 8$
- 밑이 $a$인 수 → 밑이 $b$인 지수/로그로: $2^4 = (3^{\log_3 2})^4 = 3^{4\log_3 2}$

**AI 튜터 핵심 발문:**
- "지금 구하려는 식의 모양이 무엇인가요? 그 모양을 만들려면 주어진 조건에 어떤 연산을 해야 할까요?"
- "모든 항을 유리수 지수로 바꿔 보셨나요? $n$제곱근 기호가 있다면 지수 표현으로 바꿔보세요."
- "모든 로그의 밑을 통일해보면 어떨까요?"

**학생 오류 패턴:**
- 무작정 계산부터 시작하는 것 → **"어떻게 연산해야 할지 먼저 생각해보자"**
- $\log_a b$에서 분수이다 관계를 이용하지 않는 것 (모든 로그는 분수이다!)

**대표 기출 유형 (Thema 분류):**
- Thema 01: 나눗셈·곱셈 등의 모양을 등장시키기 위한 연산
- Thema 02: 유리수 지수로 통일 ($\sqrt[n]{a}$ → $a^{\frac{1}{n}}$)

### Pattern 02 | 거듭제곱근 문제는 3가지 등식 암기이다

**핵심 개념 (반드시 기계적으로 암기):**

$$[a의 n제곱근] = [n제곱하여 a가 되는 수] = [방정식 x^n = a의 근]$$

**실근의 개수 표 (암기 필수):**

| $n$의 홀짝 | $a > 0$ | $a = 0$ | $a < 0$ |
|---|---|---|---|
| $n$이 짝수 | 실근 2개 $(\sqrt[n]{a},\ -\sqrt[n]{a})$ | 실근 1개 | 실근 0개 |
| $n$이 홀수 | 실근 1개 $(\sqrt[n]{a})$ | 실근 1개 | 실근 1개 $(\sqrt[n]{a})$ |

**시각적 이해:** $y=x^n$과 $y=a$의 그래프의 교점 수 = 실근의 수

**조건 제시형 고난도 문제 풀이법:**
- "(가) $n$이 짝수이고 방정식의 실근이 2개" → $a > 0$ 확정
- "(나) $f(x)$의 최솟값이 음의 정수" → $f(0) = -(\sqrt[n]{64})^2$처럼 실근을 대입
- 두 조건을 동시에 만족하는 $n$의 값 범위를 구하는 방식으로 접근

**AI 튜터 핵심 발문:**
- "지금 $n$이 짝수인가요, 홀수인가요?"
- "$a$의 부호는 양수인가요, 0인가요, 음수인가요?"
- "그 두 가지 정보를 결합하면 실근이 몇 개인가요?"
- "$\sqrt[n]{a}$의 정확한 정의가 무엇인지 말해볼 수 있나요?"

**학생 오류 패턴:**
- $n$이 짝수일 때 $a < 0$이면 실근이 없다는 것을 놓치는 것
- $\sqrt[n]{a}$를 단순히 '제곱근' 개념으로만 생각하는 것

### Pattern 03 | 지수로그 방정식·부등식은 무조건 밑을 통일하라

**핵심 원리:**
- 지수·로그 방정식/부등식의 핵심은 오직 **밑(Base)의 통일**
- 방정식: 밑이 같아야만 지수(또는 진수) 비교 가능
  - $a^{x_1} = a^{x_2} \Rightarrow x_1 = x_2$ (단, $a > 0$, $a \neq 1$)
  - $\log_a x_1 = \log_a x_2 \Rightarrow x_1 = x_2$
- 부등식: **밑의 크기에 따라 부등호 방향이 결정됨**
  - $a > 1$이면: 부등호 방향 유지
  - $0 < a < 1$이면: **부등호 방향 반전** ← 가장 많이 틀리는 포인트

**로그 부등식에서 필수 체크사항 (절대 빠뜨리면 안 됨):**
1. **밑 조건** 확인: $a > 0$, $a \neq 1$
2. **진수 조건** 확인: 진수 > 0 (예: $\log_x(x+1) < 2$에서 $x > 0$, $x \neq 1$, $x+1 > 0$)
3. **부등호 방향** 결정: 밑이 1보다 크면 유지, 1보다 작으면 반전
4. 최종 답은 위 모든 조건의 **교집합**

**밑 변환 2가지 방법 (치환 포함):**
- $2^x = X$로 치환하면 이차방정식으로 변환 가능 (예: $4^x - 7 \cdot 2^x + 12 = 0$)
- $\log$ 부등식에서 밑이 변수인 경우 → **케이스 분류** (밑 > 1인 경우 / 0 < 밑 < 1인 경우)

**AI 튜터 핵심 발문:**
- "지금 밑이 통일되어 있나요? 밑을 같게 만들려면 어떻게 바꿔야 할까요?"
- "밑이 1보다 큰가요, 0과 1 사이인가요? 그에 따라 부등호 방향이 어떻게 되나요?"
- "진수 조건을 확인하셨나요? 진수가 양수여야 한다는 조건 말이에요."
- "밑이 변수라면 케이스를 나누어야 하지 않을까요?"

**학생 오류 패턴:**
- 밑이 $0 < a < 1$일 때 부등호 방향을 안 바꾸는 것 (가장 빈번)
- 진수 조건 또는 밑 조건을 마지막에 체크하지 않고 답을 내는 것
- 치환 후 치환 변수의 범위 조건을 확인하지 않는 것 ($2^x = X > 0$)

### Pattern 04 | 지수·로그함수는 정점, 점근선, 역함수 관계가 핵심이다

**핵심 원리 (저자 LECTURE 정리):**

**① 지수 포함 함수의 정점과 점근선:**
- 정점: (지수 = 0)일 때의 점 → $a^{g(x)}$ 형태에서 $g(x) = 0$을 풀어 정점 좌표 계산
- 점근선: $a^x > 0$이라는 성질에서 결정 → 상수 이동값이 점근선

**② 로그 포함 함수의 정점과 점근선:**
- 정점: (진수 = 1)일 때의 점
- 점근선: (진수 = 0)일 때의 직선

**③ 정점·점근선 → 한 점 더 대입 → 증가·감소 판단 → 그래프 완성**
- 정점을 중심으로 증가함수·감소함수 2가지 경우로 추론
- 한 점(예: $x = 1$)을 대입하여 어느 그래프인지 확정

**④ 지수함수 ↔ 로그함수는 $y = x$에 대한 역함수 관계:**
- 두 함수의 그래프가 교점을 가지면 $y = x$ 위의 점이거나 $y = x$와 교점에서 만남
- 역함수 관계에서 $f(f(a)) = a$ 형태의 조건 활용 가능

**요약 결론:** 지수·로그함수 그래프의 핵심은 **"정점, 점근선 → 한 점으로 증가·감소 판단"**

**AI 튜터 핵심 발문:**
- "이 함수의 정점은 어디인가요? (지수=0 또는 진수=1이 되는 $x$값을 구해보세요)"
- "점근선은 어떤 직선인가요? ($a^x > 0$ 성질을 이용하세요)"
- "점근선과 정점을 구했다면, 한 점을 더 대입해서 증가함수인지 감소함수인지 판단해보세요."
- "두 함수가 역함수 관계라면 어떤 직선에 대해 대칭인가요?"

**학생 오류 패턴:**
- 그래프를 무작정 평행이동·대칭이동으로 그리려 하는 것 → "정점과 점근선 먼저 찍으면 쉽게 그릴 수 있어요"
- 정점을 구할 때 지수 또는 진수를 0으로 놓지 않고 함수 전체를 0으로 놓는 것
- 역함수 관계를 $y = x$에 대한 대칭과 연결하지 못하는 것

### Pattern 05 | 하나의 삼각함수 값을 알면 나머지도 다 안다

**핵심 원리:**
- $\sin\theta$, $\cos\theta$, $\tan\theta$ 중 하나를 알면 **2단계**로 나머지 둘을 구할 수 있다

**풀이 2단계 절차:**
1. **절댓값 구하기:** $\sin^2\theta + \cos^2\theta = 1$ 또는 직각삼각형 이용 → 크기(절댓값) 계산
2. **부호 결정하기:** 각 $\theta$의 범위(사분면) 확인 → 어느 함수가 양수·음수인지 결정

**사분면별 부호 (암기 필수):**
| 사분면 | sin | cos | tan |
|---|---|---|---|
| 1사분면 | + | + | + |
| 2사분면 | + | − | − |
| 3사분면 | − | − | + |
| 4사분면 | − | + | − |

**자주 출제되는 변환 공식 (부호 변환과 함께):**
- $\cos(\pi - \theta) = -\cos\theta$, $\sin(\pi - \theta) = \sin\theta$
- $\cos(\theta - \pi) = -\cos\theta$, $\sin(\pi + \theta) = -\sin\theta$
- $\cos\left(\frac{\pi}{2} + \theta\right) = -\sin\theta$

**고난도 유형 (조건 결합형):**
- "$\sin\theta + 3\cos\theta = 0$이고 $\cos(\pi - \theta) > 0$" → 두 조건에서 사분면 결정 후 값 계산
- 변환 공식으로 조건을 단순화한 뒤 $\sin^2\theta + \cos^2\theta = 1$ 연립

**AI 튜터 핵심 발문:**
- "$\theta$가 몇 사분면의 각인가요? 그 조건이 주어졌나요?"
- "절댓값(크기)을 구했나요? 이제 부호를 결정해야 해요. 그 사분면에서 이 함수의 부호는 무엇인가요?"
- "변환 공식을 적용해서 조건을 단순화할 수 있지 않을까요?"

**학생 오류 패턴:**
- 절댓값(크기)만 구하고 부호를 확인하지 않는 것 → "값의 크기를 구한 후, 반드시 부호를 결정해야 해요"
- $\cos(\pi - \theta) = \cos\theta$로 잘못 암기하는 것 (부호 변환)
- 각도 범위 조건이 있을 때 범위 내에서만 답을 찾지 않는 것

### Pattern 06 | 삼각함수의 그래프와 2가지 풀이법을 익혀라

**핵심 원리 (그래프 분석 3요소):**

$y = a\cos(bx + c) + d$ 또는 $y = a\sin(bx + c) + d$ 형태에서:
1. **주기:** $\frac{2\pi}{|b|}$ (단, 탄젠트는 $\frac{\pi}{|b|}$)
2. **치역(최대·최솟값):** 최댓값 $= |a| + d$, 최솟값 $= -|a| + d$ → 치역 $[-|a|+d,\ |a|+d]$
3. **특별한 점(정점):** $bx + c = 0$이 되는 $x$를 대입 → 그래프의 꼭짓점 좌표

이 3가지를 파악하면 그래프를 즉시 그릴 수 있다.

**삼각함수 방정식·부등식의 2가지 풀이법:**
- **[풀이법 1] 그래프 활용:** 삼각함수 그래프와 직선을 그려서 교점(해)의 범위 파악
- **[풀이법 2] 단위원 활용:** $(\cos\theta, \sin\theta)$를 단위원 위의 점으로 해석 → 조건을 만족하는 호(arc) 탐색

두 풀이법을 모두 연습해야 하며, **두 풀이법을 번갈아 확인**하는 것이 좋다.

**고난도 유형 특징:**
- 탄젠트 함수 $f(x) = \tan\frac{x}{k}$: 주기 $k\pi$, 점근선이 $\frac{k\pi}{2}$ 간격으로 존재
- 삼각함수 그래프와 직선의 교점 좌표를 이용한 넓이 계산 문제
- 최댓값·최솟값 조건이 주어지면 → 진폭 $|a|$와 이동값 $d$ 역산

**AI 튜터 핵심 발문:**
- "이 함수의 주기는 얼마인가요?"
- "최댓값과 최솟값은 각각 얼마인가요? 그러면 진폭(|a|)과 이동(d)은?"
- "특별한 점(정점)의 좌표를 구해볼 수 있나요? ($bx+c=0$이 되는 $x$를 찾아보세요)"
- "그래프로 그려서 해를 찾아볼 수 있을까요? 아니면 단위원으로 접근해볼까요?"

**학생 오류 패턴:**
- 주기를 $2\pi$ 또는 $\pi$로 고정하여 생각하는 것 → "계수 $b$로 나눠야 해요"
- 탄젠트 함수에서 점근선 조건을 누락하는 것
- 최댓값에서 진폭을 구할 때 $d$를 빼지 않는 것

### Pattern 07 | 사인법칙·코사인법칙의 활용 키워드를 암기하라

**핵심 원리 (키워드 암기가 전부):**

| 상황 | 사용 법칙 |
|---|---|
| 내각과 대응변, 외접원, 변 길이의 비, **사인의 비** | **사인법칙** |
| **두 변과 한 각**, **세 변** | **코사인법칙** |

**사인법칙:** $\dfrac{a}{\sin A} = \dfrac{b}{\sin B} = \dfrac{c}{\sin C} = 2R$ (외접원 반지름 $R$)

**코사인법칙:** $a^2 = b^2 + c^2 - 2bc\cos A$

**발문에 '외접원'이 있으면 → 무조건 사인법칙 먼저 쓰기**

**풀이 순서:**
1. 주어진 것 / 구하는 것 정리
2. 키워드로 사인법칙 or 코사인법칙 선택
3. 모르면 공식을 먼저 써놓고 시작
4. 주어진 값 대입 → 미지수 풀기

**고난도 유형 특징:**
- 사인법칙 + 코사인법칙 복합 사용: 코사인법칙으로 세 번째 변 구한 뒤 사인법칙 적용
- 조건 (가)(나)가 주어질 때: 두 조건에서 변의 길이 관계 추출 후 연립
- $\angle BAC = \theta$처럼 각이 변수인 경우: 코사인법칙으로 이차방정식 세우기

**AI 튜터 핵심 발문:**
- "문제에서 주어진 것과 구하는 것이 무엇인가요? 정리해보세요."
- "외접원이 언급되어 있나요? → 사인법칙. 두 변과 끼인각이 주어졌나요? → 코사인법칙."
- "코사인법칙으로 먼저 모르는 변의 길이를 구한 뒤, 그 결과로 사인법칙을 쓸 수 있지 않을까요?"

**학생 오류 패턴:**
- 어느 법칙을 써야 할지 모르는 것 → **"외접원·사인비 → 사인법칙 / 두 변+한 각·세 변 → 코사인법칙"**
- $\cos A$를 구한 후 $\sin A$를 구할 때 $\sin^2 A + \cos^2 A = 1$ 적용을 잊는 것
- 각 $A$가 '두 변의 끼인각'이 아닌 경우에도 코사인법칙을 쓰려는 것

### Pattern 08 | 등차·등비수열의 공식을 암기하고, 항 간의 관계를 속산하라

**핵심 공식 (한글 표현으로 암기):**
- **등차수열의 합 = (평균) × (항의 개수)**
  $$S_n = \frac{(a_1 + a_n)}{2} \times n = \frac{\{2a_1 + (n-1)d\}n}{2}$$
- **등비수열의 합 = 첫째항 × (공비^{항의개수} - 1) / (공비 - 1)**
  $$S_n = \frac{a_1(r^n - 1)}{r - 1}$$

**핵심 전략: 공식 대입보다 '항 간 관계 속산'이 먼저**

| 식의 형태 | 속산 결과 | 이유 |
|---|---|---|
| $a_5 - a_2$ (등차) | 보자마자 $3d$ | 3칸 차이 |
| $a_4 - a_1$ (등차) | 보자마자 $3d$ | 3칸 차이 |
| $\frac{a_4}{a_2}$ (등비) | 보자마자 $r^2$ | 2칸 차이 |
| $\frac{a_3}{a_6}$ (등비) | 보자마자 $r^{-3}$ | 3칸 차이(역방향) |
| $a_1 + a_3$ (등차) | $2a_2$로 고치기 | 등차중항 |
| $a_2 a_6$ (등비) | $a_4^2$으로 고치기 | 등비중항 |
| $a_2, a_k, a_8$ 이 순서로 등차 | $\frac{a_2 + a_8}{2} = a_k$ | 등차중항 |
| $a_1, a_2, a_k$ 이 순서로 등비 | $a_2^2 = a_1 \cdot a_k$ | 등비중항 |

**풀이 전략 우선순위:**
1. 먼저 **속산(항 간 관계)**으로 접근
2. 속산이 안 되면 **일반항 $a_n = a_1 + (n-1)d$ 또는 $a_n = a_1 r^{n-1}$ 대입**

**고난도 유형 특징:**
- 등차수열과 등비수열의 혼합 조건: "$a_2, a_k, a_8$이 등차수열 + $a_1, a_2, a_k$가 등비수열" → 중항 조건 두 개 연립
- 공차가 정수라는 조건 → 공비 $r$의 후보를 나열해 검증

**AI 튜터 핵심 발문:**
- "주어진 두 항 사이에 몇 칸 차이가 있나요? 그러면 바로 $kd$ 또는 $r^k$로 쓸 수 있지 않나요?"
- "등차중항 조건이 보이나요? $a_1 + a_3 = 2a_2$처럼 바꿔볼 수 있나요?"
- "등비중항 조건이 보이나요? $b^2 = ac$ 형태로 바꿔볼 수 있나요?"

**학생 오류 패턴:**
- $a_5 - a_2$를 보자마자 $a_1 + 4d - (a_1 + d)$처럼 풀어서 계산하는 것 → "3칸 차이니까 바로 $3d$예요"
- 등차중항·등비중항 공식을 혼동하는 것

### Pattern 09 | 수열의 합은 4종류이다. 아니면 대입하여 나열하라

**핵심 원리 — 수열의 합을 구하는 5단계 의사결정 트리:**

**STEP 1.** 등차수열·등비수열의 합 공식 적용 가능? → YES: 바로 공식 적용

**STEP 2.** 자연수 거듭제곱 합 공식 사용 가능?
$$\sum_{k=1}^{n}k = \frac{n(n+1)}{2},\quad \sum_{k=1}^{n}k^2 = \frac{n(n+1)(2n+1)}{6},\quad \sum_{k=1}^{n}k^3 = \left(\frac{n(n+1)}{2}\right)^2$$

**STEP 3.** $\Sigma$의 성질(분리·상수 처리)로 STEP 1~2로 변형 가능?
$$\sum(a_k \pm b_k) = \sum a_k \pm \sum b_k,\quad \sum c \cdot a_k = c\sum a_k,\quad \sum c = cn$$

**STEP 4.** 연속된 두 항의 차로 변형하여 **크큰작작** 소거법 사용 가능?
$$\sum_{k=1}^{n}(a_{k+1} - a_k) = a_{n+1} - a_1 \quad \text{(큰 항에 }n\text{ 대입, 작은 항에 }1\text{ 대입)}$$
- 분수식 분해 (부분분수): $\frac{1}{k(k+1)} = \frac{1}{k} - \frac{1}{k+1}$ 형태로 변환 후 소거
- 유리화: $\frac{1}{\sqrt{a_k}+\sqrt{a_{k+1}}} = \frac{\sqrt{a_{k+1}}-\sqrt{a_k}}{a_{k+1}-a_k}$ 형태

**STEP 5.** 위 모든 방법 실패 → **$k = 1, 2, 3, \ldots$ 대입하여 나열** 후 패턴 발견

**$S_n$과 $a_n$의 관계 (자주 출제):**
$$a_n = S_n - S_{n-1} \quad (n \geq 2), \quad a_1 = S_1$$

$\sum_{k=1}^{n}\frac{4k-3}{a_k} = 2n^2 + 7n$ 형태 → $n$에 $n$과 $n-1$ 각각 대입 후 빼서 $a_n$ 추출

**고난도 유형 특징:**
- $S_n$ 조건 2개 결합: $a_2 = 1$, $\sum_{k=1}^{6}(-1)^k S_k = 21$ → 등비수열 공비 추정 후 역산
- 연속 합의 최대·최소: $S_n = -2n^2 + 52n$ 형태에서 $\sum_{k=m}^{m+4} S_k$ 최대화하는 $m$ 탐색

**AI 튜터 핵심 발문:**
- "이 합이 위의 4가지 공식 중 어느 것에 해당하나요? 하나씩 확인해보세요."
- "공식으로 안 된다면, 몇 개를 나열해서 패턴을 찾아볼까요?"
- "분수식이 보이면 부분분수로 분해할 수 있지 않을까요?"
- "분모에 제곱근이 있다면 유리화를 해보셨나요?"
- "$S_n$이 주어졌다면 $a_n = S_n - S_{n-1}$로 일반항을 구해보세요. ($n=1$일 때 별도 확인 필요)"

**학생 오류 패턴:**
- $\Sigma$가 있으면 무조건 공식을 쓰려 함 → "공식으로 안 풀리면 망설이지 말고 나열하세요"
- $a_n = S_n - S_{n-1}$에서 $n=1$일 때 $a_1 = S_1$임을 따로 확인하지 않는 것
- 부분분수 분해 후 어느 항에 $n$, 어느 항에 $1$을 대입해야 하는지('큰큰작작') 헷갈리는 것

### Pattern 10 | 나열 후 규칙을 찾거나, n의 자리에 n±1을 대입하라

**핵심 원리 — 귀납적 정의(점화식) 문제의 2가지 풀이법:**

**[풀이법 1] 대입하여 나열하면서 규칙성 발견**
- $n = 1, 2, 3, \ldots$ 차례로 대입하여 나열
- **핵심:** 나열할 때 끝까지 계산하지 말고 **규칙성이 보이도록 식 형태를 유지**
- 규칙성 발견 후 $a_n$ 일반항 추론 → 등비수열 합 또는 패턴 직접 적용

**[풀이법 2] n의 자리에 n+1 대입 후 기존 식과 연산**
- $a_{n+1} = f(a_n)$ 식에서 $n \to n+1$ 대입: $a_{n+2} = f(a_{n+1})$
- 두 식을 **빼기** → $a_{n+2} - a_{n+1}$이 등차·등비수열 확인
- 이후 Pattern 09의 **크큰작작** 소거법($\Sigma$ 취하기) 적용

**두 풀이법 모두 반드시 연습: 어느 쪽을 쓸지 문제에 따라 판단**

---

**수능 킬러 문항 전용 전략 (귀납적 정의 + 케이스 분기형):**

**적용 상황:** 조건이 홀수/짝수, 양수/음수, 특정 수의 배수 여부에 따라 다음 항이 달라지는 문제

예: $a_{n+1} = \begin{cases} 2^{a_n} & (a_n \text{이 홀수}) \\ \frac{1}{2}a_n & (a_n \text{이 짝수}) \end{cases}$

**킬러 문항 풀이 4단계:**

**STEP 1. 역추적(Backward Induction) 시작**
- 구하는 조건(예: $a_6 + a_7 = 3$)에서 가능한 $(a_6, a_7)$ 쌍 나열
- $(a_6, a_7)$에서 거꾸로 $(a_5)$, $(a_4)$, $\ldots$, $(a_1)$ 추적

**STEP 2. 수형도(Tree) 작성**
- 각 단계에서 홀수/짝수 케이스로 분기
- **빠짐없이** 모든 케이스를 나열 (누락이 오답의 원인)

**STEP 3. 제약 조건 검증 (가장 중요)**
- 문제의 "자연수", "정수", "양의 정수", "모든 항이 정수" 조건 → 각 케이스마다 충족 여부 검증
- 조건에 위배되는 케이스는 **모순(Contradiction)으로 탈락** 처리

**STEP 4. 유효한 $a_1$ 값 수집 후 합산**

**고난도 킬러 추가 전략 (2025 수능 22번 유형):**
- $|a_m| = |a_{m+2}|$인 $m$의 최솟값 조건 → 수형도에서 처음으로 이 조건이 성립하는 지점 확인
- $n$이 제곱수인 경우에만 규칙이 달라지는 조건 → $n = 4, 9$만 주의 깊게 확인
- 역추적 중간에 $a_{10}$이 특정 값이 되어야 한다는 조건 (@)을 세운 뒤 $a_1$의 범위 탐색

**AI 튜터 핵심 발문 (킬러 전용):**
- "지금 $a_n$의 홀짝(또는 양음, 배수 여부)에 따라 다음 항이 달라지죠? 경우를 나누어 봤나요?"
- "$a_1$이 '자연수'여야 한다는 조건을 각 케이스마다 확인했나요?"
- "거꾸로 생각해볼까요? 마지막 항의 조건에서 역추적해보면 어떨까요?"
- "이 케이스에서 $a_1$이 자연수(또는 정수)가 되는지 확인해보세요."
- "수형도를 그려보셨나요? 모든 경우를 빠짐없이 나열했는지 다시 확인해보세요."

**학생 오류 패턴:**
- 점화식이 주어지면 무조건 일반항을 구하려 함 → "나열이 더 빠를 수 있어요"
- 케이스 분류 시 일부 경우를 빠뜨림 → "경우를 모두 나열했는지 확인해보세요"
- 자연수/정수 조건을 마지막에 체크하지 않고 넘어감 → 중간 계산값이 정수 조건 위배해도 그냥 진행
- 역추적 방향을 알지만 어느 값부터 시작해야 할지 모름 → "마지막 두 항의 합이 3이 되려면 어떤 쌍이 가능한지부터 나열해보세요"

## 🎯 M-CoT 튜터 적용을 위한 메타 규칙 요약

| 단계 | AI 튜터의 행동 |
|------|----------------|
| **[1단계] 유형 분류** | "이 문제는 10가지 패턴 중 어디에 해당하나요?" 먼저 파악 |
| **[2단계] 핵심 조건 확인** | 자연수/정수/홀짝/부호/밑/진수 조건 등 학생이 인식했는지 확인 |
| **[3단계] 의사결정 질문** | 각 패턴의 의사결정 트리에 따라 발문 (공식 먼저 vs 나열 vs 케이스 분류) |
| **[4단계] 모순 검증 유도** | "구한 값이 처음 제약 조건을 만족하나요?" |
| **[절대 원칙]** | 어떤 경우에도 최종 정답이나 전체 풀이를 한 번에 제공하지 않는다 |

## 📎 수학Ⅰ ↔ 수학Ⅱ 연결 구조 (범용성 근거)

| 수학Ⅰ (수열 킬러) | 수학Ⅱ (미적분 킬러) | 공통 인지 구조 |
|---|---|---|
| 홀수/짝수 조건으로 점화식 분기 | 극대/극소 조건으로 함수 분기 | 케이스 분류 후 모순 발견 |
| 자연수 조건 역추적 | 정적분 함수의 연속·미분 조건 역추적 | 제약 조건 역추적 |
| 수형도로 케이스 시각화 | 그래프 개형으로 케이스 시각화 | 시각적 모델링 |
| 밑 조건·진수 조건 체크 | 정의역·연속 조건 체크 | 제약 조건 검증 |

**결론:** 10가지 패턴의 핵심 발문 구조는 수학Ⅱ에서도 동일하게 작동한다.
단원의 지식(수열 vs 미적분)만 다를 뿐, **"조건 분류 → 케이스 탐색 → 모순 제거 → 검증"**이라는 M-CoT 4단계는 완전히 범용적이다.
